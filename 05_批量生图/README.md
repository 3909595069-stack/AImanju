# 批量生图：把文字变成画面

> 先出图，再生视频——这是 AI 短片工作流里最容易被跳过、也最不该被跳过的一步。

## 🔍 这是啥？

**批量生图**，就是根据 [[../04_分镜脚本/README|分镜脚本]] 中的每个镜头描述，逐张生成对应的静态画面。

这一步的产物是：一组和你的分镜一一对应的关键帧图像。它们是下一步 [[../06_图生视频/README|图生视频]] 的输入。

> 核心逻辑：先确保每一帧的画面是对的，再让它们动起来。跳过这一步直接文生视频 = 每一秒都在掷骰子。

## 💡 为啥做？

**为什么先出图再生视频——而非直接文生视频？**

1. **精确度**：文生视频的不可控性远大于文生图。先出图可以逐张验收画面是否对，不对就重来——成本极低
2. **一致性**：同一个角色的定妆照可以在所有生图中作为参考图锚定。文生视频没有这个锚点
3. **首尾帧控制**：图生视频可以指定首帧和尾帧，实现精确的镜头衔接。文生视频做不到
4. **试错成本**：一张图生成 4 次选最优 ≈ 10 秒；一段视频生成 4 次选最优 ≈ 20 分钟

> 一句话：生图是你对最终画面的最后一次精确控制。

## 🛠️ 怎么做？

### 生图提示词万能公式

```
[画风关键词] + [场景母代码] + [角色外貌] + [动作/表情] + [景别/角度] + [光影/色调]
```

每个模块按重要性从左到右排列——AI 对越靠前的词越敏感。

### 完整示例

```
日式动画电影风格，赛璐珞渲染，深夜便利店内部，L型空间，
左侧3排货架，门口红色立式冰柜，冷白荧光灯顶光，门口暖黄灯光渗入，
25岁女性，黑色长直发齐刘海，白色衬衫深蓝半裙，
站在收银台后面，低头看手机，嘴角微扬，
中景，平视，固定机位，冷白+暖黄+红点缀色调
```

### 画风关键词速查表

| 风格方向 | 正向提示词组合 | 推荐模型/平台 |
|:--|:--|:--|
| **日式动画** | Japanese animation style, cel shading, Ghibli-inspired, clean lines, soft gradients | 即梦 4.0 / Midjourney / Niji |
| **韩式网漫** | Korean webtoon style, manhwa art, digital painting, smooth rendering, romantic palette | 即梦 4.0 / Midjourney / Niji |
| **3D 卡通** | 3D rendered, Pixar-style, soft lighting, round shapes, subsurface scattering, C4D | 即梦 / Midjourney / ComfyUI+SDXL |
| **古风水墨** | Chinese ink wash painting, gufeng, watercolor texture, misty atmosphere, scroll painting | 即梦 / Midjourney / ComfyUI+专用LoRA |
| **写实电影** | photorealistic, cinematic lighting, 35mm film grain, anamorphic lens, shallow depth of field | Midjourney / Stable Diffusion+Realistic模型 |
| **赛博朋克** | cyberpunk, neon lights, rain-slicked streets, holographic displays, Blade Runner aesthetic | Midjourney / SDXL / ComfyUI |

### 必加负面提示词

```
丑陋, 畸形, 扭曲的手, 多余的手指, 融合的手指, 六根手指,
不对称的眼睛, 不对称的脸, 模糊, 低画质, 水印, 文字, 签名,
画框, 边框, 过曝, 过暗, 像素化, jpeg artifacts,
不同的服装, wrong clothing, 多个人, crowd, group, nude, nsfw
```

> **关于手**：把手指相关的负面词放在最前面。所有 AI 生图模型的共同弱点就是手指——这是共识，不是你的问题。

## 🐛 手指扭曲解决方案

这是最常见且最恼人的问题。以下是经过验证的解决方案，按推荐度排序：

| 方案 | 操作 | 成功率 | 成本 |
|:--|:--|:--|:--|
| **策略1：藏手** | 设计构图时让人物手在画面外/口袋里/背后/被物品遮挡 | 100% | 牺牲部分表现力 |
| **策略2：构图规避** | 不用手部特写景别，用中景以上——远景里的手 AI 能应付 | 90% | 需调整分镜 |
| **策略3：负面提示词** | 负面词加满手指相关词条，并在正向词中写 `perfect hands, 5 fingers` | 60% | 零成本尝试 |
| **策略4：多次生成** | 同 prompt 生成 8 张，赌一张手是对的 | 70%（8张中1张对） | 时间成本 |
| **策略5：后期修图** | 用 Photoshop 或美图秀秀手动修手指，修好后再喂回 AI | 100%（但费时） | 修图时间 |
| **策略6：局部重绘** | 在 ComfyUI 中对手的区域进行 inpainting，只重绘手部 | 85% | 需要 ComfyUI 环境 |

> **实用主义建议**：能藏就藏，能避就避。在观众不会注意的地方纠结手指完美度，是典型的"创作者自嗨"。
