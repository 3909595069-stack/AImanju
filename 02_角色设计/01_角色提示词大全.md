# 🎨 角色设计提示词大全

## 万能提示词公式

```
[性别][年龄段] + [脸型] + [发型+发色] + [眼睛特征] + [肤色]
+ 穿着[服装描述] + [配饰] + [气质/表情倾向]
+ [画风关键词] + [光线] + 纯白背景 + 角色设定图 + 三视图/全身站立
```

## 关键提示词指令速查

| 用途 | 中文 | 英文 |
|------|------|------|
| 三视图 | 三视图、正面+侧面+背面 | `three views`, `character turnaround`, `front side back` |
| 角色设定图 | 角色设定图、角色设计图 | `character design sheet`, `model sheet`, `reference sheet` |
| 多表情 | 表情包、多表情 | `expression sheet`, `multiple expressions` |
| 全身像 | 全身站立 | `full body standing`, `full length` |
| 纯色背景 | 纯白背景、纯色背景 | `white background`, `plain background` |
| 多角度 | 多角度展示 | `multiple angles`, `multiple views` |

## 分步生成法（最可靠）

由于 AI 对「一张图精准排布多个视角」的控制有限，推荐 **先分别生成再拼合**：

| 步骤 | 操作 |
|------|------|
| 1 | 生成正面全身图 |
| 2 | 复制提示词，将「正面」改为「侧面朝向（身体朝右）」，生成侧面 |
| 3 | 改为「背对镜头、背面站立」，生成背面 |
| 4 | 单独生成面部特写近景 |
| 5 | 用 PS / Canva / PPT 拼合为三视图卡片（16:9 横向） |

### 三视图排布模板

```
┌──────────────────────────┬──────────────────────────────┐
│   面部特写（左侧 50%）     │  正视图 │ 侧视图 │ 背视图   │
│   五官细节 + 微表情        │ （右侧三等分排列）            │
├──────────────────────────┴──────────────────────────────┤
│  底部信息栏：角色名 | 身高 | 年龄 | 配色色卡 (10%)          │
└─────────────────────────────────────────────────────────┘
```
