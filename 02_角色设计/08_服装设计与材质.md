# 👗 服装设计与材质描述

> 服装不只是"穿什么"——它告诉观众这个角色的身份、阶层、性格和当下的状态。

## 服装材质词汇库（AI 提示词专用）

在 AI 提示词中，**材质描述比款式名称更重要**。AI 不理解"时尚穿搭"，但它理解材质的光学属性：

| 材质 | 英文关键词 | 视觉特征（告诉AI的关键） |
|------|-----------|------------------------|
| **丝绸/缎面** | `silk, satin` | 柔和高光(glossy finish), 流动垂坠感(fluid drape) |
| **皮革** | `leather, distressed cowhide` | 细腻纹理(fine grain), 微油光泽(subtle oily sheen) |
| **天鹅绒** | `velvet` | 深邃柔光(deep soft reflection), 哑光奢华(luxurious matte) |
| **棉麻** | `linen, cotton` | 哑光颗粒感(matte granular texture), 自然漫反射 |
| **牛仔布** | `denim, distressed denim` | 做旧纹理(worn texture), 撞色明线(contrast topstitching) |
| **蕾丝** | `lace` | 精致镂空花纹(delicate pattern), 半透明(semi-transparent) |
| **羊毛/针织** | `wool, knitted, cable knit` | 厚重绞花纹理(thick cable knit texture) |
| **薄纱/欧根纱** | `tulle, chiffon, organza` | 轻薄半透明(lightweight translucent), 飘逸(ethereal) |
| **金属/盔甲** | `metallic, oxidized copper, chrome, brushed steel` | 镜面反射/拉丝/锈蚀铜绿 |
| **皮草** | `fur, faux fur` | 蓬松柔软(fluffy soft texture), 毛流方向(fur direction) |

## 服装属性三层描述法

在写提示词时，按三层结构描述服装：

```
第一层·版型：立领 / V领 / 圆领 / 中式交领 / A型 / H型 / 收腰 / 及膝
第二层·材质：丝绸光泽 / 棉麻哑光 / 皮革纹理 / 金属反光
第三层·特异性锚点：左胸徽章是金色鹰 / 右手袖口有磨损 / 腰带扣是方形银扣
```

### 完整提示词示例

```
# 版型 + 材质 + 锚点

立领黑色风衣，领口翻起（版型），
暗色羊毛混纺面料，低调哑光质感（材质），
左侧领口别一枚银色飞鸟胸针（锚点），
风衣下摆微微磨损（锚点）。

# 古风版

交领右衽白色长袍，宽袖（版型），
真丝提花面料，流动光泽，袖口有银线云纹刺绣（材质+锚点），
腰间束一条红色编织腰带，挂一枚圆形象牙玉佩（锚点）。
```

## 服装与场景/状态联动

角色在不同的剧情状态下，服装应该有相应的变化：

| 剧情状态 | 服装变化 | 提示词关键词 |
|---------|---------|-------------|
| 战斗后 | 破损+脏污 | `torn sleeve`, `frayed edges`, `dirt stains`, `blood spatter` |
| 雨天 | 湿透+贴身 | `wet fabric clinging to skin`, `damp darkened by rain` |
| 情绪紧张 | 褶皱增加 | `tensed fabric`, `stretched seams`, `wrinkled collar` |
| 逃亡中 | 凌乱+丢失配饰 | `disheveled hair`, `missing one earring`, `coat half-buttoned` |
| 正式场合 | 盛装+精致 | `formal attire`, `pristine condition`, `tailored fit` |

## 服装一致性锁定技巧

### "材质参数收敛法"

不要用简单词描述衣服（如 `leather jacket`），用具体的材质参数：

❌ 简单的：`leather jacket`
✅ 具体的：`distressed cowhide texture, fine grain, subtle oily sheen, edge creases at elbows, dark brown with reddish undertone`

这样 AI 每次理解的都是同一个"材质指纹"，不会变来变去。

### "IP-Adapter 服装锚定法"

1. 生成第一帧满意服装 → 截图保存服装局部
2. 后续分镜生图时，把服装局部图放入即梦参考图
3. 参考强度设为 0.4-0.6（只锚定服装，不影响姿势和表情）
