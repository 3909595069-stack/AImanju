# 林夕 — 主角完整形象设计

> 基于真人照片 + 性格设定（务实主义实干型 / 规则破坏者 / 观众之眼 / 粘合剂）  
> 视觉风格：半写实动画质感，新海诚风格光影，9:16竖屏

---

## 长相

身高177cm，体型属于"看着不显但动起来有劲"的类型——不壮不瘦，肩膀宽度刚好撑得起T恤，腰线偏窄。电气工程实验课经常搬器材，手臂有紧实的肌肉线条但不夸张。站姿微微前倾——不是驼背，是那种"随时准备下一步动作"的姿态。

**脸型**：介于鹅蛋脸和方脸之间的"少年感轮廓"——下颌线条清晰但不锋利，下巴偏尖但不尖锐。脸颊饱满度刚好，不婴儿肥也不瘦削。从侧面看，鼻梁到下巴的线条流畅干净。整张脸最大的特点是"干净"——不是帅得惊天动地，而是五官分布均衡、比例舒服、越看越顺眼的那种。

**眉毛**：眉毛是整张脸最有辨识度的部位。浓密但不杂乱，眉形偏平直，眉尾微微上扬大约5度——刚好卡在"认真"和"不太在乎"之间。不说话的时候眉毛会微微往中间聚，给人一种"这人在想事情"的感觉。同文说他"眉毛比嘴会说话"。

**眼睛**：偏圆的杏眼，瞳色是深棕色——日常光下近乎黑，阳光下会变成琥珀色。双眼皮但不宽，眼尾微微下垂大约3度——这就是为什么他明明在认真想事情，看起来却像在发呆。眼神干净直接，看人的时候不闪躲也不审视，就是"我在听，你说"。但当他想通一件事的时候，眼睛会突然亮起来——从"发呆模式"切换到"搞事模式"的开关就在眼睛里。顺城说他的眼睛"像猫——平时懒洋洋的，亮起来你就知道他要闯祸了"。

**鼻子**：鼻梁挺直，鼻头偏小且微微上翘。侧面线条干净。鼻梁上有一道很浅的横痕——不是疤，是长期戴眼镜的压痕。他不戴眼镜了，但痕迹还在。偶尔思考问题的时候会不自觉用食指推一下鼻梁上并不存在的眼镜——瑞樟第一次看到这个动作时沉默了两秒，然后说："你需要一副眼镜。"林夕："我不近视。"瑞樟："……我知道。"

**嘴唇**：厚度适中，上唇薄下唇稍厚。嘴角在放松状态下微微向下——不是不开心，就是"默认表情"。但笑起来的时候嘴角会上扬得比一般人更多，形成强烈的反差——"不笑的时候像在生气，笑起来像另一个人"。门牙有一颗稍微突出一点点，不明显，但笑大了能看到，给他增加了一种"不太精致但很真实"的质感。

**皮肤**：偏白但不是苍白——在宿舍打代码和在校园里跑来跑去之间的平衡色。脸上干干净净，没有痘痘也没有刻意保养的痕迹。右脸颊靠近耳根处有一颗很小的痣——不仔细看发现不了。

**头发**：纯黑色，发质偏硬偏直。长度在"该剪了"和"还能撑两天"之间——刘海刚好到眉毛上方，两侧头发盖住半个耳朵，后颈的头发有一点翘。不是刻意做的造型，就是自然生长+顺手扒拉的结果。风吹过的时候头发会乱，但他从来不整理——同文每次看到都要伸手帮他扒拉一下，他每次都说"不用"，但从来不躲。

## 穿着

**设计原则**：林夕的穿着要同时体现三个关键词——务实（不买没用的东西）、理工男（实用主义审美）、随性（懒得想搭配）。不能潮——他不是那种会研究穿搭的人。不能土——毕竟是年轻人，基本的审美在线。是在"我随便穿的"和"但看着还行"之间的精准平衡。

### 日常装束

**上装**：深灰色重磅棉T恤（圆领，领口刚好卡在锁骨位置）。不是那种薄的透肉的T恤——220克以上的重磅棉，挺括有型，洗了十次也不变形。左胸口有一个极小的口袋——不是装饰，他真的会在里面放一支笔。T恤袖子刚好包住上臂的二头肌中段——不长不短。

天冷的时候外面套一件藏青色连帽卫衣——不是运动品牌，是学校周边商店买的那种印着"武夷学院"四个小字的普通卫衣。帽子边缘有一点褪色——洗太多次了。卫衣口袋边缘磨出了毛边。这件卫衣是他穿得最久的衣服，从大一开学穿到现在。

**下装**：黑色直筒工装裤——不是修身款也不是阔腿款，刚好在两者之间。膝盖位置有两个贴袋——他真的会在里面放东西（U盘、螺丝刀、实验室门禁卡）。裤脚微微卷起一道边，露出脚踝。裤子的面料是那种耐脏耐磨的棉混纺，洗了多次后呈现出自然的做旧感。

**鞋子**：灰白色低帮帆布鞋——不是名牌，就是淘宝六七十块的那种。鞋带是后来换的深蓝色鞋带（原配的白色鞋带断了）。鞋头有一点磨损，鞋底边缘有一些实验室地板上蹭到的灰色痕迹。穿的时候鞋带系得很松——他习惯不弯腰直接把脚踩进去。

**配饰**：左手戴一只黑色Casio基础款电子表——不是G-Shock，是更便宜的那种方形的学生表。表带是树脂的，表盘上有几道轻微的划痕。这个表是他高中毕业时买的，用了快两年。功能只有一个：看时间。林夕："能看时间就行，其他功能手机都有。"

右手中指第一关节有一枚很素的银色戒指——不是饰品，是他大一时做电路实验，第一次亲手焊成功的PCB板上的一个小金属环。他把它打磨后套在了手指上。大小刚好卡在关节上不掉。瑞樟注意到这枚戒指后问过——林夕只说"焊的第一块板子"。瑞樟没有再问。但这枚戒指在后续剧情中会多次被特写——它是林夕作为"电气工程学生"的身份锚点，也是"规则破坏者"的隐喻：把电路板上的东西变成身体的一部分。

**背包**：黑色帆布双肩包——不是电脑包，是那种最简单的抽绳款帆布包。包里永远是三样东西：笔记本电脑、充电宝、一瓶水。包底有一小块被笔油染蓝的痕迹——洗不掉，他也不在意。背带上用黑色马克笔写了一个很小的"夕"字——是大一刚入学时为了防止和室友的包弄混写的。

### 特殊场景变体

**熬夜/图书馆场景**：在上述基础上多一件外套——灰蓝色格子法兰绒衬衫，不扣扣子直接当外套穿。这件衬衫是他高中时穿的，现在袖口明显短了一截——露出半截手腕。领子已经洗得微微起球。他不care。

**雨天**：一把纯黑色的折叠伞——手柄上贴着一张标签，写着他的名字和电话号码（他丢过两把伞之后贴的）。穿一双深蓝色塑胶拖鞋去上课，到了教室再换帆布鞋。拖鞋就放在教室门口。同文第一次看到时笑了三分钟。林夕："湿鞋子上课难受。我不在乎别人怎么看我。"

**"正式"场合**（如果剧情需要他穿得正式一点）：他只有一件浅蓝色牛津纺衬衫，长袖，扣子扣到第二颗。配的还是那条工装裤和帆布鞋。衬衫是面试/答辩专用——平时压在背包最底层，拿出来的时候总是皱的。他不熨。同文帮他熨过一次，他问"有什么意义，穿上五分钟就皱了"。同文回："那你为什么还要吃饭，吃完五个小时就饿了。"

## 色彩锚定

| 部位 | 颜色 | 材质 | 功能 |
|------|------|------|------|
| T恤 | 深灰(#3a3a3a) | 220g重磅棉 | 低调、耐脏、不用想搭配 |
| 卫衣 | 藏青(#2c3e50) | 全棉毛圈 | 保暖、"武夷学院"身份标签 |
| 裤子 | 黑(#1a1a1a) | 棉混纺斜纹 | 耐脏耐磨、口袋实用 |
| 鞋子 | 灰白(#e8e8e0) | 帆布橡胶 | 便宜好走、不心疼 |
| 手表 | 黑+灰(#1a1a1a) | 树脂 | 只看时间 |
| 戒指 | 银(#c0c0c0) | 不锈钢 | 第一块PCB——身份锚点 |
| 背包 | 黑(#1a1a1a) | 帆布 | 三件套：电脑+充电宝+水 |
| 法兰绒衬衫 | 灰蓝(#6b7b8d) | 法兰绒 | 高中穿到现在——念旧但不承认 |
| 牛津衬衫 | 浅蓝(#a8c8e0) | 牛津纺 | 只有面试答辩穿——皱的 |

**色彩设计逻辑**：
- 全身不超过黑白灰蓝四个颜色——不用想搭配
- 没有红色/橙色/黄色——这些是"需要被注意"的颜色，林夕不需要被注意
- 唯一的金属是银色——戒指，冷调，和他的务实性格一致
- 和守护者角色形成对比：瑞樟的米白金纹（光芒外露）、同文的墨绿银饰（精致感）、顺城的棕红（温暖活力）——林夕的深灰黑蓝是"沉默的背景色"，但正好是这种沉默让他成为团队的底色

## 外貌辨识度设计（"一眼认出林夕"的三个标记）

1. **右手中指的银色戒指**：第一个镜头特写就可以给到。他做实验、拿手机、拿筷子——每一次手部动作都在强化这个视觉标记。观众会在第三集左右开始注意到"他手上那个戒指"。
2. **永远不整理的头发**：刘海刚好在眉毛上方，风一吹就乱，他从来不整理。同文伸手帮他扒拉——这是他们关系的视觉锚点。
3. **推不存在的眼镜**：思考时食指推鼻梁上的空气。瑞樟注意到→给他建议→他不接受——两个角色性格差异用一个动作讲完。

## 与角色性格的呼应

| 性格特质 | 穿着体现 |
|----------|----------|
| 务实主义 | 全身不超过四个颜色，基础款，不买没用的东西 |
| 理工男思维 | 工装裤贴袋装U盘/螺丝刀、戒指是第一块PCB、Casio便宜表 |
| "规则破坏者" | 戒指是电路板零件——把"工具"变成"身体的一部分"；衬衫袖子短一截但继续穿 |
| 不在乎别人看法 | 雨天拖鞋上课、背包有笔油不洗、不熨衬衫 |
| 念旧但不承认 | 高中穿到现在的法兰绒衬衫、大一买的表、第一次焊的PCB |
| 随性但不随便 | 220g重磅棉（有要求但不外露）、帆布鞋鞋带换过颜色（暗戳戳的小心思） |

### 日常表情切换
