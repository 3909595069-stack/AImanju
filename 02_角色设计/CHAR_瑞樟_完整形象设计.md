# 瑞樟 — 完整形象设计

> 身份：瑞樟教学楼地标化身，内含六大学院（人文/生态/机电/数计/土木/艺术）  
> 性格：外表冷峻的理性主义学者，内心给每个同伴写诗的温柔灵魂  
> 视觉风格：半写实动画质感，新海诚风格光影，9:16竖屏

---

## 长相

身高183cm，体型清瘦但不单薄，属于"坐图书馆三个月但每天早上跑五公里"的那种瘦——骨架开阔，肩宽腰窄，穿什么都撑得起来。站姿笔直但不僵硬，重心微微偏右脚，双手习惯性交叠在胸前或一只手插在裤袋里。走路时步幅偏大但速度不快——不是在赶路，是在思考。

**脸型**：偏长但不尖，颧骨位置偏高，下颌线条清晰但不锋利。从正面看脸部轮廓干净利落，从侧面看下颌到下巴的线条像数学公式一样精确。脸上没有多余的肉——不是因为瘦，是因为骨骼结构本身就撑起了整个轮廓。这张脸不适合笑，但笑起来意外地好看。

**眉毛**：浓而不粗，眉形偏平直，眉尾微微上挑约3度。不是同文那种"我知道你不知道"的锐利挑眉，而是一种更安静的、像在批改论文时的"这里有问题"的微挑。眉心偶尔会出现一道极浅的竖纹——不是皱眉，是长期思考形成的痕迹。

**眼睛**：丹凤眼，深褐色瞳色。眼型偏细长，内眼角尖锐外眼角微微上挑。看人的时候有一种"正在把你当成一道数学题解"的专注感——让你既紧张又莫名想被他多看两眼。睫毛偏长但不卷翘，垂下眼时睫毛会在脸上投下浅浅的阴影。不近视但戴眼镜——他说"眼镜是一种结界"。

**眼镜**：银色半框眼镜，镜片极薄。镜框上缘是金属银色，下缘无框。镜腿细长，末端有很浅的防滑纹理。这副眼镜是他化形后自己"长出来"的——不是从石头里，是从他意识到"我需要一台过滤器"的那一刻出现在脸上的。摘掉眼镜的瑞樟看起来年轻五岁，眼神也会变得没那么有距离感。但他几乎从不摘。

**鼻子**：鼻梁高挺，鼻尖偏尖，鼻翼窄。从侧面看鼻梁线条笔直，没有任何弧度——像用尺子画的。鼻尖在光线好的时候能看到一点很淡的高光。

**嘴唇**：嘴唇偏薄，唇形清晰。上唇的唇峰明显但不尖锐，下唇比上唇稍薄。不笑的时候嘴角微微向下，看起来永远在思考什么重大问题。笑起来的时候整个人的气质完全变了——眼睛弯成月牙，脸颊出现两个很浅的酒窝，高冷人设瞬间崩塔。但这两个酒窝只出现在他真的被逗笑的时候——不是礼貌性微笑，是忍不住的那种。

**皮肤**：偏白但不是苍白，是"长期室内但偶尔会在茶山上吹风"的透亮质感。脸部皮肤干净，没有痘印或瑕疵。右手食指有一道浅浅的疤痕——做土木实验时不小心被混凝土试块划的。他没当回事，但这个细节会在很多手部特写镜头里出现。

**头发**：黑茶色，在阳光下能看出茶色的层次。微微自然卷，不是那种明显的卷，是有几缕头发在刘海处打了个很浅的弧度。长度刚好到眉毛上方，平时不怎么打理但就是不乱——属于让室友咬牙切齿的那种天生好发质。发质偏细偏软，风吹的时候不像林夕的头发那样乱飘，而是一整片微微波动——像风吹过茶山的茶叶。

**耳朵**：耳型端正，耳垂偏小。左耳耳廓上有一个极小的缺口——不是伤疤，是天生的小缺口。同文第一个发现的："你耳朵缺了个角。"瑞樟："……那是天然公差。"同文："？"瑞樟："制造误差。万物皆有不完美之处。"同文："你耳朵缺个角你说那是制造误差？？？"

## 穿着

**设计原则**：瑞樟的穿着要体现三个关键词——学院派（不是学校的学院，是知识分子气质）、精确（衣服合身到毫米级）、隐藏的浪漫（平时看不出来，特定时刻才会显现的细节）。

### 日常装束

**上装**：米白色（接近象牙白）亚麻混纺衬衫。不是纯棉——亚麻混纺意味着有天然的肌理感和微微的褶皱感，不是邋遢的皱，是"这件衣服有呼吸"的质感。领子是标准的尖领，但第一颗扣子从不扣——不是同文那种"故意不扣"，是他觉得扣上会限制思考。袖口永远卷到手肘以上两指位置——卷得很整齐，像用尺子量过。左右袖口卷的高度完全一致。衬衫口袋里永远插着一支黑色中性笔——笔夹朝外，笔尖朝下。笔是那种文具店三块钱一支的普通款，但里面的笔芯是他自己换的0.38mm细芯。

**内搭**：衬衫里面是一件纯白色圆领打底T恤——只在衬衫领口敞开时能看到一截白色领口边缘。材质是轻薄的全棉，贴身穿的，别人一般看不到。但如果剧情需要他脱掉衬衫（比如战斗后衬衫破了），里面的白T恤就是最朴素的内搭。

**下装**：深灰色修身长裤，但不是紧身——站着的时候裤管笔直，走路的时候膝盖处有自然的布料褶皱。面料是羊毛混纺，有垂感。口袋是斜插式的，里面只装三样东西：手机（屏幕永远朝内）、一张折叠的A4纸（用来临时写公式）、一支备用笔。左右口袋的东西完全对称。

**皮带**：深棕色皮质，针扣式，不是自动扣。皮带宽度约3cm，皮质偏硬，用了很久但保养得很好，表面有一层自然的包浆光泽。针扣的针孔只有一个有正常使用的痕迹——他每次系在同一个孔。同文说这很瑞樟。

**鞋子**：棕色皮质帆布鞋——不是运动帆布鞋，是皮质鞋面+帆布鞋底的那种牛津鞋和帆布鞋的杂交款。系带式，鞋带是深棕色的扁蜡绳。鞋底偏薄，走路的声音比林夕的帆布鞋更清脆——每一步都有节奏，像节拍器。

**配饰**：
- **左手戴一块老式机械表**——银白色表盘，棕色皮质表带，罗马数字时标，表盘直径约38mm，不大不小刚好贴合手腕。不是名牌，但机芯声音极好听——凑近能听到清脆的每秒五下的擒纵声。他说这是"身体里土木工程那部分灵魂的坚持——在一切都可以数字化的时代，机械不需要理由。"这块表的表带是后来换的——原配的表带在大一那年断了，他自己换了一根。新表带的颜色比原配的深一个色号。
- **偶尔戴一副细框护目镜**——只在做实验的时候出现。银色细框，镜片是透明的防护材质。和林夕说话时会把护目镜推到额头上——这是一个很好看的镜头角度。
- **不戴任何首饰**。戒指、项链、手链——一概没有。"多余的质量会增加决策变量。"同文："你耳朵缺个角也是决策变量？"瑞樟："……"

### 激活态变化

当六种学科的力量被同时激活时，他的衬衫上会浮现出流动的金色纹路——不是刺绣，是从布料内部透出的光。六种图案在布料上缓慢游走：

- **齿轮**（机电工程）——在右肩位置，缓慢自转
- **电路**（数学与计算机）——在左前臂位置，像PCB走线一样分支
- **诗句**（人文与教师）——在背部，行书字体，内容随机变化，每次都是不同的诗句
- **建筑图纸**（土木工程）——在胸口到腹部位置，是学院楼的结构蓝图线条
- **叶脉**（生态与资源工程）——在衣领和袖口边缘，像茶山茶树叶片的脉络
- **音符**（艺术）——在衣角下摆，像五线谱上的音符碎片

激活态下他的眼睛会微微泛出淡金色——不是发光，是瞳孔颜色从深褐变成琥珀金。眼镜镜片偶尔会映出他正在看的物体的结构分解图——Engineering版的X光视觉。

### 战斗状态

金色纹路的亮度提升到可以照明的程度。头发微微浮动——但不是同文那种无风自动，而是像被静电轻轻吸起几根发丝。右手掌心会出现一个金色的光核——乒乓球大小，是他六种力量的汇聚点，也是他发射力量的位置。光核出现时周围空气温度会上升约两度。

## 色彩锚定

| 部位 | 颜色 | 材质 | 功能 |
|------|------|------|------|
| 衬衫 | 象牙白(#f5f0e8) | 亚麻混纺 | 学院气质、有肌理感 |
| 打底T恤 | 纯白(#ffffff) | 轻薄全棉 | 贴身穿，只在领口露出一截 |
| 长裤 | 深灰(#4a4a4a) | 羊毛混纺 | 垂感、笔挺 |
| 皮带 | 深棕(#5c3d2e) | 皮质针扣 | 用很久、保养好 |
| 鞋子 | 棕(#6b4a3a) | 皮质帆布 | 走路有节拍感 |
| 手表 | 银白+棕(#e8e0d0) | 机械表 | 土木工程灵魂的坚持 |
| 眼镜 | 银(#c0c0c0) | 半框金属 | "结界" |
| 激活纹路 | 金(#d4a843) | 光 | 六种学科图案 |
| 光核 | 金(#e8c550) | 能量 | 右手掌心 |

**色彩设计逻辑**：
- 主色象牙白+深灰：理性和干净的底色
- 所有辅色都偏暖（棕、金）——暗示他隐藏的温柔
- 金色只出现在激活态——"他的光芒是内在的，平时不显"
- 和林夕对比：林夕的黑灰蓝是"沉默的背景色"，瑞樟的象牙白金纹是"需要被读懂的篇章"

## 外貌辨识度标记

1. **银色半框眼镜**：最明显的视觉标记。摘掉眼镜=切换人格。
2. **衬衫袖子永远卷到手肘**：两只袖口高度完全一致——强迫症级别的精确。
3. **右手食指的疤痕**：土木实验留下的。在拿笔、拿书、伸出食指指向关键信息时都会被特写。
4. **左手机械表**：镜头给到他思考时会切手表特写——听到擒纵声=他在算。
5. **左耳小缺口**：自然的"制造误差"，是他身上唯一不完美的地方。同文发现了。他没发现同文发现了。

## 与角色性格的呼应

| 性格特质 | 外观体现 |
|----------|----------|
| 六种学科共存 | 激活态六种图案在衬衫上游走，每种代表一个学院 |
| 选择困难 | 皮带只有一个孔有使用痕迹——他每次系在同一位置，但会犹豫要不要换 |
| 外表冷峻内在温柔 | 平时嘴角向下/象牙白+深灰的冷调；笑起来有酒窝/金色纹路是暖调 |
| 理性主义 | 袖子卷得完全对称、口袋物品全球对称、笔必须笔夹朝外 |
| 隐藏的浪漫 | 激活态诗句图案、酒窝、机械表的擒纵声、给每个同伴写诗 |
| 学者气质 | 眼镜、笔、A4纸、护目镜——所有配饰都是"知识工具"而非"装饰品" |
