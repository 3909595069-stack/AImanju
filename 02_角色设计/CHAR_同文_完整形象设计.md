# 同文 — 完整形象设计

> 身份：同文教学楼地标化身，内含旅游学院/商学院/海外教育学院  
> 性格：精明的外交官，朋友圈里最仗义的那个，嘴上损你但会帮你把事情办得明明白白  
> 视觉风格：半写实动画质感，新海诚风格光影，9:16竖屏

---

## 长相

身高168cm，身材比例极好——腿长占了总身高的一大半，属于那种"站在人群中不需要看脸就能认出她的轮廓"的类型。不是娇小可爱风，是走路带风型。步幅偏大，步频偏快——不是在赶时间，是"我的时间很值钱"。站在任何人旁边都有一种"我比你早到三秒"的气场。

**脸型**：标准鹅蛋脸，饱满但不圆润。额头宽度适中，颧骨位置刚好在脸部中线偏上——这就是为什么她扎高马尾特别好看。下巴弧度柔和但收得干净利落。整张脸的比例是那种"第一眼觉得舒服，第二眼觉得精致，第三眼发现很难忘"的类型。

**眉毛**：眉毛修得很精致但不过分——不是那种画出来的精致，是天然眉形好+稍微修了一下杂毛的程度。眉峰位置刚好卡在瞳孔正上方偏外约2mm——这个位置同时满足"干练的锐利感"和"不会让人觉得凶"。眉毛的颜色比头发颜色浅一个色号——深棕色偏灰。

**眼睛**：大而有神，眼尾微微上挑约5度。瞳色是偏浅的棕色——在室内是深琥珀色，到阳光下会变成半透明的浅琥珀色，能看到虹膜上的放射状纹理。双眼皮宽度适中，内眼角尖锐外眼角微微上扬——那个"还没开口你就觉得她什么都知道了"的眼神。"看人的时候不闪躲——她不是在看你的外表，是在读你的动机。"

**睫毛**：睫毛不算特别长，但密度高——垂下眼时会在下眼睑上投出一排整齐的阴影。刷睫毛但不浓，效果是"她眼睛本来就长这样"。

**鼻子**：小巧挺翘。鼻梁细直，鼻尖微翘不上翻。从侧面看鼻子在脸上的占比刚好——不是那种"鼻子太抢戏"的类型，而是"这张脸上每个器官都在自己该在的位置"。

**嘴唇**：嘴唇饱满，上唇有清晰的唇峰——不是画出来的，是天然的M型唇峰。下唇比上唇略厚约1/3。日常涂一层很薄的润唇膏，偶尔会用偏豆沙色的口红——色号偏低调，但效果是"她今天气色特别好"而不是"她今天化了妆"。说话时嘴唇的动态很有表现力——说正事时嘴唇动作精准（商人模式），吐槽时嘴唇动作会变夸张（损友模式）。

**门牙**：有一颗门牙微微比旁边的牙齿突出约1mm——非常细微，不仔细看看不出来。但笑大了就会看到——不是标准的完美笑容，但就是让人忘不了。这个特征让她的笑容从"好看"变成"有辨识度"。

**皮肤**：健康的小麦色偏浅——不是那种在海滩晒出来的小麦色，是商学院presentation站台灯下+海外教育学院field trip户外调研+偶尔在校园里走去开会之间积累出来的恰到好处的日晒感。肤色均匀有光泽。左脸颊靠近颧骨位置有几颗极浅的小雀斑——要在特定光线下才能看到。她从不遮。

**头发**：纯黑色，发质偏粗偏直——扎马尾的时候从来不需要用皮筋绕第二圈，一圈就紧。长度刚好到肩胛骨下缘——放下来的时候发尾刚好搭在肩胛骨下方。日常扎利落的高马尾——马尾位置在后脑勺正中偏上。扎马尾的皮筋是黑色的那种最小号的弹力绳，上面缠着一段极细的银色丝线——是在海外教育学院时从一个法国交换生那里学来的习惯，说"银色丝线会让马尾看起来更利落"。

**碎发**：有几缕碎发永远不太听话地散在鬓角和后颈——不是她没扎好，是这些碎发本身长度刚好卡在"扎不住"和"太短"之间。瑞樟看见了说这是"熵增定律的具象化"。同文头也不回地说:"这叫character。"这两缕碎发是她最好的表情辅助——说正事时她会有意识地把碎发别到耳后（这是她的"let's get serious"动作），吐槽时碎发散着就行。

**耳朵**：耳型小巧，耳垂圆润。左耳戴一排三个银色小耳环——耳垂一个、耳廓中段一个、耳廓上缘一个，大小递减，但都很小（最小的那个直径约2mm，不仔细看会以为是耳钉）。这三个耳环是她化形时就有的——出生的标记。右耳从不戴任何东西——"右耳用来听世界的，戴东西挡住算怎么回事。"

## 穿着

**设计原则**：同文的穿着要体现三个关键词——国际化（有质感的简洁利落，一看就知道见多识广）、锋利度（衣服的剪裁线条利落，但不会让人觉得有攻击性）、隐藏的仗义（看起来精英但其实口袋里有零食和折叠刀）。

### 日常装束

**上装**：白色真丝混纺衬衫——不是那种死板的工作衬衫，是真丝和棉的混纺面料，有真丝的光泽感和棉的挺括感。领子是标准的小翻领，第一颗扣子从不扣——不是为了性感，是"扣上就太像在开会了"。衬衫下摆塞进裤腰（前面——让腰线看起来更高），但后面稍微拉出来一点（不那么正式）。袖口是法式反折——不需要袖扣，就是简单的反折两圈，刚好到手腕上方约三指。右手袖口偶尔会蹭到一点钢笔水——她用钢笔。

**下装**：深墨绿色高腰阔腿裤。高腰设计把腰线拉到肋骨下缘——这是她168cm看起来像175cm的核心技巧。阔腿不是夸张的阔，是裤管刚好能遮住鞋面但不会拖地的程度。面料是那种有垂感的西装面料——走路的时候裤管会微微飘动但不是裙子那种飘，是"这个女人在前进"的飘。裤腰是隐藏式的侧拉链+挂钩，正面看没有多余的扣子或扣眼，干净利落。

**外套**：卡其色中长款风衣，长度到大腿中段。剪裁极其合身——肩部有极薄的垫肩，刚好让肩线看起来利落但不宽。腰部有腰带但不是用金属扣——是用风衣同面料的系带，她通常不系，就让带子自然地垂在两侧或者松松地在前面打一个单结。风衣的袖口有可调节的袖袢，总是扣在最紧的那一档——因为她的手腕偏细。风衣内侧有两个暗口袋——左边放手机，右边放名片夹。

**风衣口袋里的东西**：同文的风衣口袋是一个"你永远猜不到能掏出什么"的地方：
- 左边外口袋：一包纸巾、两颗薄荷糖（给需要的人）、一个硬币大小的便携翻译器
- 右边外口袋：折叠小刀——很小，刀柄是深色木纹，刀刃约5cm长。削苹果用的——她也确实削苹果
- 左边内暗袋：三张名片——分别印着中/英/法三种语言
- 右边内暗袋：一支钢笔（笔尖EF尖，墨水是她自己调的蓝黑色）+ 一张对折的便签纸

**内搭**：风衣里面在冷天会加一件薄款圆领羊毛衫——通常是烟灰色或深酒红色。领口刚好露出衬衫领子，袖口露出约1cm衬衫袖。

**鞋子**：黑色尖头平底鞋——尖头但不是那种很长的尖，是"刚好比你预想的多尖了1cm"的设计。鞋面是软牛皮，鞋底是橡胶的（能跑能跳），鞋跟在1.5cm左右。配黑色隐形船袜——站着看不到袜子边缘。走路的步频偏快——她的高跟鞋声音不是那种慢悠悠的嗒……嗒……嗒……，是快速的、有节奏的嗒嗒嗒嗒——像打字机。

**配饰**：
- **左耳三个银色小耳环**：出生标记。从小到大排列，耳垂那个最大(约4mm环)、耳廓中段居中(约3mm环)、耳廓上缘最小(约2mm环)。材质是某种不氧化的银色金属——不是银，是她自己身体的一部分。
- **右手无名指一枚极细的银色尾戒**：没有任何装饰，就是一个最简单的银色细环。不是婚恋相关的寓意——是商学院时导师送她的，上面刻着一个极小的词：`connect`。她每次签合同之前会用左手拇指转一下这枚戒指——这是她的习惯动作。
- **左手手腕一条极细的编织手环**：棕色和墨绿色线编织的。不是买的——是旅游学院时带学生团去东南亚，一个当地小孩送她的。手环很旧了，颜色褪了不少，但她从不摘。
- **不戴手表**。她说：手机就在口袋里，戴表是多余的。顺城说那是因为她每三分钟看一次手机。同文:"那不是看时间。那是掌握先机。"

### "进入状态"的变化

当同文切换到"谈判/交锋状态"时——不管是对外的交涉还是对内的团队决策——她的外表会有细微变化:

1. 周围空气微微波动——不是发光，是像夏天柏油马路上的热浪一样的空气折射
2. 发梢开始无风自动——只有发梢，辫子主体不动。像被静电轻轻吸引
3. 舌头自然而然地切换到对方最舒服的语言——不只是中英切换，是发音习惯的调整。和一个四川人说四川话、和一个北京人说普通话、和一个英国人用英式口音——这不是她用力的，是海外教育学院在她灵魂里刻下的印记。她甚至自己都意识不到
4. 眼睛里的琥珀色会微微加深——从浅琥珀变成深琥珀。对方如果注意到这个变化，八成已经输了

这个状态下她说过的最后一句话永远是对的。

## 色彩锚定

| 部位 | 颜色 | 材质 | 功能 |
|------|------|------|------|
| 衬衫 | 白(#fafafa) | 真丝混纺 | 光泽感+挺括，不扣第一颗扣子 |
| 裤子 | 深墨绿(#2d4a3e) | 西装面料 | 高腰阔腿，垂感，视觉增高三寸 |
| 风衣 | 卡其(#c4b59a) | 棉混纺斜纹 | 剪裁合身，肩有薄垫肩 |
| 内搭毛衣 | 深酒红(#6b3a4a)或烟灰(#888888) | 薄羊毛 | 冷天穿，领口露衬衫，袖口露1cm |
| 鞋子 | 黑(#1a1a1a) | 软牛皮 | 尖头平底，橡胶底能跑能跳 |
| 耳环×3 | 银(#c0c0c0) | 身体一部分 | 从小到大排列，右耳不戴 |
| 尾戒 | 银(#c0c0c0) | 导师赠 | 刻字`connect`,谈判前转一下 |
| 编织手环 | 棕+墨绿(#8b6914/#2d4a3e) | 棉线编制 | 东南亚小孩的礼物，褪色了 |
| 钢笔 | 蓝黑(#1a1a2e) | 自调墨水 | EF尖，名片和便签 |

**色彩设计逻辑**:
- 主色白+墨绿+卡其:清爽利落有记忆点，但不过分张扬
- 墨绿裤子是全角色体系中最有辨识度的单品——在同系列任何镜头中看到一片墨绿就知道她在场
- 所有金属都是银色——冷调，对应她"精明干练"的社会面
- 唯一的暖色是深酒红毛衣(冷天穿)——是她被冻到时才会露出的柔软面
- 和林夕(黑灰蓝)、瑞樟(象牙白金纹)形成三极色彩区分:三个角色站在一起，观众凭颜色就能认出谁是谁

## 外貌辨识度标记

1. **高马尾 + 墨绿阔腿裤**:远距离识别——整个系列中没有任何其他女性角色扎这个发色+穿这个颜色的裤子。全景镜头中只需要一个轮廓就认得出。
2. **左耳三个银色耳环**:近距离识别。瑞樟永远不戴首饰，顺城只戴红绳，同文是唯一"明确有耳洞且戴了多个耳环"的守护者。特写时这三个耳环的大小排列会成为一个令人舒服的视觉细节。
3. **右手中指的转尾戒动作**:这是她最重要的"tell"——谈判前/做重大决定前/被问到不想回答的问题时，她都会用左手拇指转一下右手的尾戒。观众第三集左右会自己发现这个规律。
4. **鬓角碎发**:永远不太听话的两缕头发。她没有"整理头发"的习惯(同文不需要照镜子)，但说正事之前会下意识把碎发别到耳后——这就是"现在开始认真了"的信号。
5. **风衣口袋里的折叠小刀**:不是武器——是"我能在任何地方削苹果"的物资储备。第一次掏出来削苹果时林夕愣了五秒。同文:"怎么了，你没见过刀?"林夕:"……我没想到你这种人会随身带刀。"同文:"你这种人又是什么人。"(把削好的苹果递过去)林夕接过来咬了第一口之后再也没有问过问题。

## 与角色性格的呼应

| 性格特质 | 外观体现 |
|----------|----------|
| 社交情报场 | 风衣口袋有中/英/法名片+翻译器+薄荷糖——为任何对话做准备 |
| 精明+仗义 | 钢笔自调墨水(精致的讲究)+小刀削苹果(给你的人是另外一回事) |
| 中英夹杂 | 名片三种语言、海外教育学院印记、法语交换生经验(马尾银丝) |
| 嘴上损你心里帮你 | 牙齿微突让笑容从"好看"变成"有辨识度"——不完美但让人忘不了 |
| 不退让 | 平底鞋但能跑能跳——不需要高跟鞋来增加气场 |
| 隐藏的脆弱 | 褪色的手环(念旧)、深酒红毛衣(冷天才出现的柔软色) |
