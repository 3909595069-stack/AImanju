# 🌍 视觉世界观构建

> 别急着生图。在做任何一个分镜之前，先回答：这个世界是什么样的？规则是什么？"好看"这个词太模糊了——你需要一个具体的、可执行的视觉体系。

## 世界观五大支柱

### 1. 时代设定
| 时代 | 核心视觉特征 | 常见题材 |
|------|------------|---------|
| 古代中国 | 木结构建筑、纸窗、烛光、石板路 | 仙侠、武侠、宫斗 |
| 近代民国 | 中西合璧、黄包车、蒸汽火车、老式电话 | 谍战、爱情 |
| 现代都市 | 玻璃幕墙、LED屏、地铁、智能手机 | 甜宠、职场、悬疑 |
| 近未来(2040-2080) | 全息广告、悬浮车、AR隐形眼镜 | 科幻、赛博朋克 |
| 架空玄幻 | 悬浮山、灵石、灵力光效、异兽 | 修仙、奇幻 |

### 2. 地理体系

画一张简单的世界地图（哪怕只是草稿），标注：
- 主角的活动范围
- 关键地点（家、学校/宗门、常去的咖啡厅/客栈）
- 地点之间的空间关系（A在B的东边，走过去要多久）

### 3. 势力与视觉符号

如果有多个阵营/门派/组织，给每个设计一套视觉符号：

| 势力 | 主色 | 标志 | 建筑风格 |
|------|------|------|---------|
| 青云宗（正道） | 白+蓝 | 云纹、仙鹤 | 悬空亭台、白玉材质 |
| 魔教（反派） | 黑+血红 | 骷髅、荆棘 | 暗黑城堡、黑铁材质 |
| 散修联盟（中立） | 灰+绿 | 树叶、流水 | 自然木屋、竹材质 |

### 4. 规则体系

- **力量体系**：这个世界的力量来源是什么？（灵力/科技/金钱/信息）
- **社会规则**：谁说了算？平民的日常生活是怎样的？
- **禁忌**：在这个世界里，什么是绝对不能做的？

### 5. 视觉风格统一

选定一种美术风格后，**全剧所有镜头都用同一种风格关键词**：

| 风格 | 关键词 |
|------|--------|
| 日式二次元 | `anime style, cel shading, clean line art` |
| 韩式条漫 | `manhwa style, webtoon, soft shading` |
| 中国古风 | `chinese ink painting, gufeng, traditional` |
| 3D动画 | `3d animation, pixar-inspired, smooth rendering` |
| 写实电影感 | `cinematic, photorealistic, film grain` |
| 赛博朋克 | `cyberpunk, neon noir, high contrast` |

## 世界观文档模板

为你的剧建立一个"世界圣经"文档，以下是要包含的内容：

```
# [剧名] 世界观档案

## 时代
- 时期：[古代/现代/未来/架空]
- 年代/年号：[具体]

## 地理
- 主城/主场景：[名称 + 简要描述]
- 关键地点列表：
  - [地点A]：[功能] + [视觉特征]
  - [地点B]：[功能] + [视觉特征]

## 势力
- [势力名]：
  - 主色：[颜色]
  - 标志：[符号]
  - 建筑风格：[描述]

## 规则
- 力量/科技体系：[描述]
- 社会结构：[描述]

## 视觉风格
- 美术风格：[统一关键词]
- 色彩基调：[主色调+辅色]
- 禁止出现的元素：[不符合世界观的元素清单]
```

## 世界观一致性的自查清单

在做每一镜之前，过一遍：

- [ ] 这个场景的时代感对吗？（古代场景有没有出现现代物品？）
- [ ] 这个场景和世界观的地理关系一致吗？（角色不能从A城瞬移到B城）
- [ ] 这个道具/服装和角色的设定一致吗？（穷人不穿名牌、古人不拿手机）
- [ ] 这个画面的美术风格和全剧一致吗？（没有突然从二次元跑到写实）
- [ ] 这个场景里的"不完美细节"是这个世界特有的吗？（仙侠的"不完美"≠赛博的"不完美"）
