# 💡 光影色调的工程化控制

> 光影不是"加个滤镜"——它是画面情绪的核心语言。同一场景中光影不一致，就像同一部电影里突然换了摄影指导。

## 光影的三个铁律

### 铁律 1：主光永远有方向

禁用平光（front light / flat lighting）。平光只适用于资产设定集和人设图。

| 光源方向 | 效果 | 适用情绪 |
|---------|------|---------|
| **45° 侧光** | 立体感最强，最常用 | 日常、对话、叙事 |
| **逆光（背光）** | 金色轮廓光，剪影感 | 浪漫、神秘、高潮 |
| **顶光** | 戏剧性强，眼窝有阴影 | 悬疑、紧张、审讯 |
| **底光** | 诡异、不自然 | 恐怖、反派登场 |
| **侧逆光（Rembrandt）** | 脸颊三角形亮光 | 高级人像、权力感 |

### 铁律 2：角色边缘必须有轮廓光

逆光勾边 = **1-2 像素的亮边**，把角色从背景中"拉"出来。没有轮廓光 → 角色融入背景 → 画面扁平。

```
提示词加入：
rim light on character, subtle edge highlight, character separated from background
```

### 铁律 3：环境光与场景色调一致

- 冷蓝色场景 → 环境光补冷蓝
- 暖黄色场景 → 环境光补暖黄
- 霓虹场景 → 环境光补对应霓虹色

**千万不要**：暖黄场景 + 冷蓝补光 = 画面"脏"，像没调色。

## 色彩锚定法

在每一镜的 Prompt 末尾，强制挂载同一个色彩风格簇：

```
# 不同风格的色彩锚定
Teal and Orange cinematic lighting（青橙电影光影）
Warm tungsten interior + cool blue moonlight（冷暖对比）
Wes Anderson pastel palette（韦斯安德森粉彩）
Blade Runner neon noir（银翼杀手霓虹黑色）
Studio Ghibli natural light（吉卜力自然光）
```

## 场景色彩方程 = 情绪公式

你的场景色彩**直接决定**观众看到这个场景时的第一情绪反应：

| 场景色彩方程 | 观众反应 |
|-------------|---------|
| 高饱和暖色(>70%) + 明亮高调 | 快乐、能量、希望 |
| 高饱和冷色(>70%) + 高对比 | 紧张、速度、科技感 |
| 低饱和暖色(<30%) + 柔光 | 怀旧、温情、回忆 |
| 低饱和冷色(<30%) + 暗调 | 悲伤、孤独、死亡 |

## 跨场景色彩过渡

如果你的剧有多个场景，场景之间的色调不能"跳"——需要平滑过渡：

| 上一个场景 | 下一个场景 | 过渡技巧 |
|-----------|-----------|---------|
| 暖色室内 → 冷色室外 | 中间加一镜：从窗户往外看（同时包含暖色室内和冷色室外） |
| 白天 → 夜晚 | 加一镜黄昏过渡（橙金色调），或直接黑屏转场 |
| 温馨 → 悬疑 | 逐渐去饱和（从 100% → 30%），暗示情绪转变 |

## 专属 LUT 工作流（进阶）

1. 在剪映/DaVinci Resolve 中针对你的漫剧定制一套 LUT（Look Up Table，色彩查找表）
2. 第一集调好色后，**导出 LUT 文件**
3. 后续所有集数都加载这个 LUT → 全系列色调统一
4. AI 生成的原始像素 → LUT 强制映射到预设色域 → 杂乱的素材瞬间拥有统一质感
