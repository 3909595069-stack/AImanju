# 🎞️ 场景转场与视觉连续性

> 镜头之间不是"切了就行"——好的转场让观众意识不到场景变了，坏的转场让观众每次都在心里"切"一下。

## 四种相似性转场（蒙太奇手法）

利用两个镜头中相似的视觉元素实现无缝过渡：

| 类型 | 原理 | AI漫剧示例 |
|------|------|-----------|
| **形状相似** | 两个具有相似几何形状的主体衔接 | 圆月 → 圆形时钟表盘；圆形镜子 → 圆睁的眼睛 |
| **运动相似** | 前一镜的运动势能在后一镜延续 | 汽车向右冲出画面 → 飞鸟向右飞翔；挥剑劈砍 → 瀑布倾泻 |
| **位置相似** | 主体固定在画面同一位置 | 主角瞳孔在画面中央 → 星球漩涡与瞳孔重叠 |
| **概念相似** | 同一核心主题串联不同画面 | "破碎"串联：玻璃破碎 → 照片撕碎 → 瞳孔裂纹 |

## 六种实用转场手法

### 1. 遮挡转场（最易实现）

利用画面中物体短暂遮挡镜头形成自然间隙来切换：

```
镜头A：角色身体靠近填满画面 → 短暂全黑 → 镜头B：下一个场景出现
```

### 2. 动作匹配转场

```
镜头A：门关上（向前运动）→ 镜头B：另一扇门打开（同一方向运动）
```

### 3. 听觉引导转场

画面切换前0.5秒，先出现下一个场景的声音：

```
镜头A：女主在卧室（画面）+ 镜头B的海浪声渐入 → 切 → 镜头B：女主在海边
```

### 4. 推轨/拉轨替代硬切

```
镜头缓慢推向角色眼睛 → 瞳孔模糊 → 从瞳孔后拉 → 角色已在新场景
跟拍角色背影穿过走廊 → 背景在行走中渐变为另一时空
```

### 5. 语义动词触发渐变（AI专属技巧）

在分镜提示词末尾添加过渡指令：

> *"镜头随女主脚步穿过拱门→门洞边缘像素缓慢溶解→溶解完成时已置身竹林小径"*

### 6. 色彩过渡

```
暖色室内 → 中间加一镜窗户往外看（同时包含冷暖两色）→ 冷色室外
白天 → 黄昏（橙金过渡）→ 夜晚
```

## 镜头组接五大铁律

| 原则 | 解释 | 怎么操作 |
|------|------|---------|
| **景别渐进** | 跳跃不超过两个层级 | 远景→全景→中景→近景→特写，不要特写直接跳远景 |
| **动接动/静接静** | 运动镜头接运动镜头，固定接固定 | 如果上一镜是推镜头，下一镜也应该是运动镜头（或中间加一镜固定过渡） |
| **不越轴** | 视线方向保持同一侧 | 两人对话时，A看右B看左 → 整场戏不变 |
| **30°原则** | 同主体切换至少改变30°角度 | 如果两个镜头拍同一个人，角度变化<30°=看起来像剪辑失误 |
| **匹配原则** | 动作/视线/图形在剪接点匹配 | 手伸向门把→切→手抓住了门把（动作在剪接点完成） |

## "世界圣经"文档

为你的整个故事世界建立一份**永远不会变**的参考文档：

| 模块 | 内容 |
|------|------|
| 时代设定 | 古代/现代/未来/架空 + 具体年代或时期特征 |
| 地理体系 | 主场景地图、关键地点、场景间的空间关系 |
| 势力体系 | 门派/组织/阵营 + 各自的视觉符号（颜色/标志/建筑风格） |
| 规则体系 | 力量体系/科技层级/社会规则 |
| 视觉风格 | 统一的美术基调、色彩体系、材质语汇 |

> 💡 **自查标准**：每次设计转场时问自己——"上一镜最后一个画面和下一镜第一个画面有什么逻辑关联？观众能否在0.3秒内意识到这是同一个故事？"
