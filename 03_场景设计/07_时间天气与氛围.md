# 🌤️ 时间、天气与场景氛围

> 同样一个地点，早上8点和凌晨3点是完全不同的两个世界。时间+天气=场景的情绪基调。

## 一天中的时间情绪表

| 时间 | 光线特征 | 色温 | 情绪 | 适合场景 |
|------|---------|------|------|---------|
| **清晨 5-7点** | 柔和的蓝调晨光，薄雾，长阴影 | 5000-6500K（偏冷） | 宁静、希望、新开始 | 主角起床、出发、晨练 |
| **上午 8-11点** | 明亮的自然光，阴影清晰 | 5500K（中性） | 日常、活力、工作 | 大部分日常场景 |
| **正午 12-2点** | 顶光强烈，阴影短而硬 | 5500-6000K | 压迫、曝晒、对峙 | 西部对决、烈日下的战斗 |
| **午后 2-5点** | 光线柔和偏暖，阴影拉长 | 4500-5000K（偏暖） | 温馨、惬意、休闲 | 咖啡厅、约会、回忆 |
| **黄昏/黄金时刻** | 金色柔光，逆光剪影，天空橙粉 | 3000-4000K（很暖） | 浪漫、怀旧、告别、决定性时刻 | 表白、分别、高潮 |
| **蓝调时刻** | 日落后20分钟，深蓝色天空 | 8000-10000K（很冷） | 孤独、忧郁、神秘 | 内心独白、悬疑 |
| **夜晚** | 月光/路灯/霓虹，暗部占多数 | 取决于光源 | 神秘/浪漫/危险 | 夜戏、悬疑、动作 |

## 天气的情绪语言

| 天气 | 视觉特征 | 情绪 | AI漫剧场景 |
|------|---------|------|-----------|
| **晴天** | 高对比度，蓝天，清晰阴影 | 快乐、活力、正常 | 日常对话、开心场景 |
| **阴天** | 低对比度，灰白天空，柔阴影 | 平淡、压抑、沉思 | 内心戏、过渡场景 |
| **雨天** | 湿润反光，冷调，雨丝 | 悲伤、浪漫、净化、新生 | 分手戏、转折点 |
| **暴雨** | 强对比，水花，昏暗 | 紧张、危险、冲突爆发 | 战斗、追逐、对峙 |
| **雪天** | 白茫，柔光，高调 | 纯洁、孤独、终结 | 结局、内心平静 |
| **雾天** | 低能见度，柔化轮廓，神秘 | 悬疑、迷茫、未知 | 恐怖、悬疑、梦境 |
| **沙尘暴** | 橙黄调，颗粒感，压抑 | 末日、绝望、生存 | 末世场景 |

## 场景氛围切换技巧

### 同一个地点，不同时间

```
# 白天版本
Coffee shop interior, bright afternoon sunlight through windows, warm atmosphere, soft shadows

# 夜晚版本（所有非时间关键词完全一致）
Coffee shop interior, dim warm pendant lights, dark outside windows, intimate atmosphere, 
soft yellow glow, same brick walls, same wooden floor, same vintage cash register
```

> 💡 **关键技巧**：切换时间/天气时，只改变光源和氛围词，**所有空间结构描述全部保持不变**。

### 季节变化

```
# 春天
Cherry blossom avenue, pink petals falling, soft spring sunlight, fresh green leaves

# 秋天
Cherry tree avenue, red and golden leaves covering the ground, warm autumn sunlight, 
bare branches visible, swirling dry leaves in wind
```

## 天气转场作为叙事工具

天气变化 = 情绪变化。不用台词告诉观众"气氛紧张了"，用一个天气变化：

| 叙事需求 | 天气变化 |
|---------|---------|
| 冲突即将爆发 | 晴天 → 乌云密布 |
| 真相即将揭露 | 雾天 → 雾散（视野变清晰） |
| 角色内心绝望 | 开始下雨 |
| 角色获得新生 | 雨后彩虹 / 雪后天晴 |
| 时间流逝 | 同一个地点的四季更替快剪 |
