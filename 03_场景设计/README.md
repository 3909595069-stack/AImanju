# 场景设计：给空间上一个"母代码"

> 角色的舞台不能每场戏都在换——场景设计就是给环境写一个"不会变的底本"。

## 🔍 这是啥？

**场景设计**，不是画一张好看的背景图。而是为每个故事空间定义一套**环境母代码**——一组固定参数，确保同一个地点在所有镜头里看起来是同一个地方。

类比理解：
- 角色设计 = 身份证，锁定"人"
- 场景设计 = 房产证，锁定"空间"

如果你在第三幕回到第一幕的咖啡馆，观众应该一眼认出那就是同一家咖啡馆——而不是另一个"看起来像咖啡馆的地方"。

## 💡 为啥做？

两个核心原因：

1. **空间连续性**：AI 每一张图都是独立生成的，今天的咖啡馆和明天的咖啡馆在 AI 眼里没有任何关系。你需要主动锁定。

2. **以景叙事**：场景不是"背景板"，它是叙事的一部分：
   - 凌乱的房间 → 主角刚失恋
   - 一尘不染的办公室 → 角色的强迫症性格
   - 黄昏的旧站台 → 离别情绪的视觉化

> 好的场景设计让观众不用看字幕就知道故事的氛围。

## 🛠️ 怎么做？

### 场景母代码：四要素锁定法

为每个主要场景建立一张"母代码卡"，包含以下四个要素：

| 要素 | 说明 | 示例（深夜便利店） |
|:--|:--|:--|
| **标志性物体** | 这个空间独一无二的物件，是"地标" | 门口的红色立式冰柜、柜台上的招财猫、墙上褪色的"欢迎光临"贴纸 |
| **光源与时间** | 主光源方向 + 色温 + 时间设定 | 顶光冷白荧光灯（色温 4000K）+ 门口暖黄路灯（色温 2700K）从玻璃门渗入，时间：凌晨 2:00 |
| **色调体系** | 场景的主色调 + 辅色调 + 强调色 | 主：冷白色（灯光）；辅：暖黄色（门口路灯渗光）；强调：红色（冰柜/商品包装） |
| **空间结构** | 关键的空间关系（什么东西在什么位置） | L 型空间：进门左侧 3 排货架，正面是收银台，右侧靠窗有一排 4 个座位，最里面是员工通道门 |

### 场景母代码模板

```
场景名：[场景名称]
标志物：[2-3个独一无二的物体/细节]
光源：[方向] [类型] [色温]，时间：[具体时段]
色调：主[颜色] + 辅[颜色] + 强调[颜色]
空间：[关键空间关系描述]
```

### 实际使用流程

1. 为每个场景建一张母代码卡
2. 生图时，将母代码的各项翻译为提示词：
   ```
   深夜便利店内部，L型空间，左侧3排货架，
   门口红色立式冰柜，柜台招财猫，
   冷白荧光灯顶光，门口暖黄灯光渗入，
   冷白+暖黄+红点缀色调
   ```
3. 同一场景的所有分镜都使用同一组母代码——只改变视角和景别，不改变四要素

## 📊 光影情绪对照表

| 情绪 | 光源类型 | 色温 | 色调方向 | 示例场景 |
|:--|:--|:--|:--|:--|
| **温馨/幸福** | 暖黄光，多点漫反射 | 2700-3200K | 橙黄 + 奶油白 + 浅木色 | 家庭晚餐、咖啡馆午后、黄昏客厅 |
| **紧张/悬疑** | 单点硬光，强阴影 | 偏冷或偏绿 | 暗蓝 + 深灰 + 一点红 | 深夜走廊、废弃建筑、审讯室 |
| **悲伤/孤独** | 低照度柔光，大面积暗部 | 偏蓝 5000-6500K | 蓝灰 + 冷白 + 极淡暖色作对比 | 雨天窗边、空房间、凌晨街头 |
| **史诗/宏大** | 强烈逆光，Rayleigh 散射 | 金色时刻 3500K | 金黄 + 深褐 + 天空蓝 | 日落战场、山巅远眺、大教堂内 |
| **科幻/未来** | 霓虹/屏幕光，多色混光 | 混合色温 | 青蓝 + 品红 + 黑 | 赛博朋克街道、太空舱、实验室 |

> **使用技巧**：把上表中的色温和色调词直接写进提示词，AI 对光影的理解比你想象的要敏感得多。

## 🐛 做坏了怎么办？

| 问题 | 症状 | 解决方案 |
|:--|:--|:--|
| 场景漂移 | 同一场景在不同镜头里像不同地方 | 固定场景描述模板，每个该场景的提示词都用同一段开头 |
| 光影打架 | 光源方向不统一，影子到处飞 | 在提示词中明确指定"单光源，从[方向]打来"，不要写多个光源 |
| 色调跑偏 | 暖色调场景突然出了冷色图 | 在负面提示词加入相反色调关键词；在正向提示词首句强调色调 |
| 空间结构崩塌 | AI 把空间布局搞错了 | 加上空间方向词（L型、U型、开放空间）；尝试用 ControlNet 的深度图控制 |
