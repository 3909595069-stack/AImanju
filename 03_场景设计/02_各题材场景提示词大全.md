# 🎨 各题材场景提示词大全

> 万能公式：`{风格定位}，{场景主体}，{时间+天气}，{氛围情绪}，{细节元素}，{镜头+光影}，高细节，电影级渲染`

## 一、古风/仙侠场景

### 核心关键词库

| 类别 | 关键词 |
|------|--------|
| 风格 | 国漫古风、仙侠、水墨风、敦煌壁画风、中国风 |
| 建筑 | 飞檐翘角、亭台楼阁、悬浮宫殿、古桥、寺庙 |
| 自然 | 竹林、云雾、灵石、雪山、瀑布、松柏 |
| 装饰 | 龙纹、云纹、金色琉璃瓦、白玉栏杆 |
| 氛围 | 空灵、庄严、苍茫、静谧、飘逸 |

### 场景示例

**仙侠洞府疗伤：**
```
国漫仙侠风，昏暗的洞府石壁，深夜，零星灵石发出微光，
缠绕绷带的石床，周围漂浮紊乱的灵气光点，
电影级侧逆光，浅景深，氛围凝重，高细节
```

**江南烟雨：**
```
水墨风国漫，江南古街，清晨，细雨绵绵，
青石板路湿润反光，两侧白墙黛瓦阁楼，
挂着油纸伞的商铺，雾气缭绕，
低角度平拍，柔和光影，温馨静谧
```

**云间仙宫：**
```
敦煌壁画风，悬浮在云端的宫殿，正午晴空万里，
周围环绕龙凤祥云，宫殿梁柱雕刻龙纹，
金色琉璃瓦，阳光穿透云层形成光柱，
广角仰拍，宏伟庄严
```

**雪山侠客：**
```
写实古风，雪山之巅，黄昏夕阳染红冰河，
孤独剑客背影，背后插古朴宝剑，
周围凛冽风雪，岩石裸露，
低角度仰拍，戏剧化光影，孤寂苍茫
```

**竹林抚琴：**
```
云雾缭绕的竹林深处，白衣公子坐在石凳上抚琴，
琴弦微动，竹叶随风飘落，
画面静谧悠远，柔和光影，东方美学
```

**月下煮茶：**
```
山顶凉亭，月光皎洁，白发老者煮茶，
古琴在侧，茶汤冒着热气，
远处连绵群山，宁静祥和，江湖隐士氛围
```

### 古风负面提示词
```
--no 现代汽车、霓虹灯、塑料材质、现代建筑、电线杆、空调外机
```

## 二、赛博朋克/科幻场景

### 核心关键词库

| 类别 | 关键词 |
|------|--------|
| 风格 | 赛博朋克、未来科幻、蒸汽朋克、废土 |
| 环境 | 霓虹灯光、全息投影、悬浮车、高楼霓虹、雨夜 |
| 细节 | 湿滑街道、蒸汽管道、齿轮机械、全息广告 |
| 色调 | 蓝紫主调、品红点缀、高对比度、霓虹反射 |
| 氛围 | 压抑、密集、科技感、混乱、冷酷 |

### 场景示例

**赛博雨夜追逐：**
```
赛博朋克风，未来都市小巷，深夜暴雨，
霓虹灯光（红/蓝/紫）反射在湿滑路面，
两侧挂满全息广告的墙体，低空飞行悬浮车，
手持相机动态跟拍，高对比度，氛围紧张
```

**未来实验室：**
```
科幻风，高科技实验室，清晨，室内冷白光，
全息投影屏幕显示数据，银色金属实验台，
机器人助手，悬浮实验样本，
顶光+侧光，浅景深，严谨冷静氛围
```

**废土科幻：**
```
末日赛博风，废弃未来都市，黄昏沙尘暴，
倒塌高楼框架，裸露钢筋管道，
破旧悬浮车残骸，橙黄色调，
广角俯拍，光影杂乱，荒芜压抑
```

**量子便利店：**
```
2077年街头，全息投影便利店招牌闪烁，
半机械人店员，眼球切换扫描模式，
悬浮无人配送舱掠过，霓虹蓝紫色调
```

**机械花海：**
```
未来废弃都市，生锈机械花瓣风中转动，
白色连衣裙仿生人蹲在花海中，
指尖轻抚机械花瓣，眼底模拟人类的温柔泪光
```

### 科幻负面提示词
```
--no 古代建筑、木质结构、油纸伞、自然森林、烛光、马车
```

## 三、现代/校园/都市场景

### 核心关键词库

| 类别 | 关键词 |
|------|--------|
| 风格 | 二次元卡通、写实现代、日系、韩漫风 |
| 地点 | 校园操场、教室、便利店、天台、咖啡厅 |
| 细节 | 樱花树、霓虹招牌、车流光影、课桌、自动贩卖机 |
| 色调 | 清新暖色、自然日光、温暖治愈 |
| 氛围 | 青春、日常、温馨、孤独、治愈 |

### 场景示例

**校园樱花：**
```
二次元卡通风，高中校园樱花大道，午后晴天，
粉色樱花飘落，学生穿校服走过，
阳光透过樱花形成斑驳光影，
平拍视角，柔和暖色调，青春温馨
```

**都市天台：**
```
写实现代风，城市天台，黄昏夕阳西下，
远处高楼林立天际线，天台有休闲座椅和盆栽，
微风拂动窗帘，广角俯拍，暖橙色调，治愈氛围
```

**深夜便利店：**
```
二次元风格，街角便利店，深夜小雨，
店内暖光透出，门口霓虹招牌闪烁，
街道空无一人，湿滑地面反射灯光，
中景平拍，冷色与暖色对比，孤独静谧
```

**咖啡厅角落：**
```
日系现代风，街角咖啡厅，午后暖阳靠窗座位，
阳光洒在木质桌面，冒热气的咖啡杯，
窗外行人模糊，浅景深，温馨治愈
```

## 四、奇幻/魔幻场景

### 场景示例

**精灵森林：**
```
奇幻风，发光精灵森林，夜晚满月，
荧光藤蔓缠绕古树，蘑菇发出蓝绿色微光，
精灵飞舞留下光迹，林间雾气，
低角度仰拍，柔和月光，梦幻氛围，动态粒子效果
```

**巨龙战斗：**
```
史诗奇幻，乌云翻涌天空，深夜雷电交加，
黑色巨龙盘旋飞舞，口中喷吐火焰，
下方残破古堡废墟，火焰照亮云层，
手持相机跟拍，高对比度，震撼
```

**水晶矿洞：**
```
魔幻风，地下水晶矿洞，无自然光，
彩色水晶发出璀璨光芒，地面散落水晶碎片，
矿道两侧发光矿石，侧逆光勾勒水晶轮廓，
浅景深，神秘梦幻
```
