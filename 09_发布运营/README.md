# 发布运营：让作品被看到

> 费了那么大力气做出来，不发出去就不是作品——它只是一个文件。这一步，把文件变成作品。

## 🔍 这是啥？

**发布运营**，不是"按一个发布按钮"。它包括：

| 子环节 | 内容 | 影响 |
|:--|:--|:--|
| **封面设计** | 定帧+标题的组合，决定观众点不点 | 点击率的核心变量 |
| **标题撰写** | 3秒内传达"这讲什么+为什么你要看" | 和封面共同决定点击 |
| **文案/描述** | 补充信息、引导互动、标签策略 | 影响推荐算法的分发 |
| **发布时间** | 选对时段，事半功倍 | 影响初始曝光量 |
| **多平台分发** | 一个作品多平台发布，但策略要适配 | 决定最大触达范围 |

## 💡 为啥做？

一个残酷的事实：**在短视频平台上，你的作品有 1-3 秒的时间决定观众是点进去还是划走。**

- 封面 + 标题 = 那 1-3 秒里唯一能被看到的东西
- 内容再好，封面和标题不行 = 零点击 = 零播放 = 白做
- 这不是"包装"，这是给你的作品一个公平竞争的机会

> 把 90% 的精力花在制作上，再把 10% 的精力花在让这 90% 被看到——那 10% 决定了一切。

## 📊 多平台分发策略

| 维度 | 抖音 | B站 | 小红书 | 快手 |
|:--|:--|:--|:--|:--|
| **最佳时长** | 30秒-2分钟 | 2-5分钟 | 30秒-1.5分钟 | 30秒-1分钟 |
| **最佳画幅** | 9:16 竖屏 | 16:9 横屏 | 3:4 竖屏 | 9:16 竖屏 |
| **内容偏好** | 快节奏、强冲突、情绪起伏大 | 深度叙事、制作精良、有"弹幕槽点" | 审美感强、治愈/文艺、教程向 | 接地气、强冲突、反转 |
| **标题风格** | 制造好奇/争议 + 短 | 信息量 + 可以稍长（15-30字） | 实用/审美 + 加 emoji | 直白/夸张 + 短 |
| **标签数量** | 3-5 个精准标签 | 5-8 个标签 | 5-10 个标签（多用长尾词） | 3-5 个标签 |
| **最佳发布时段** | 12:00-14:00, 18:00-21:00 | 12:00-14:00, 19:00-22:00 | 7:30-9:00, 12:00-14:00, 20:00-22:00 | 12:00-14:00, 18:00-22:00 |
| **互动重点** | 评论区带节奏（自己先发第一条评论） | 引导弹幕 + 评论区讨论 | 收藏引导 + 评论区种草 | 评论区制造话题 |

### 分发建议

- **首次发布**：在一个平台首发，观察 24 小时数据，优化后再发其他平台
- **多平台适配**：不要直接把抖音竖版发到 B站——至少要重新构图/加黑边/重新字幕排版
- **不要同时发所有平台**：每个平台的推荐算法会检测"全网首发性"——间隔 2-4 小时分发

## 🎨 好封面公式

```
好封面 = 主角脸（大且有表情） + 冲突暗示（让观众产生问题） + ≤ 8 字标题（印在封面上）
```

### 封面设计原则

| 原则 | 说明 | 好例子 | 坏例子 |
|:--|:--|:--|:--|
| **主角脸 > 场景** | 人脸天然吸引注意力。封面上人脸占比 > 1/3 | 角色正脸或半侧脸特写，表情到位 | 只有风景/空镜头/背影 |
| **有冲突暗示** | 让观众产生一个问题："发生了什么？" | 一个女孩惊恐地看着手机——她在看什么？ | 一个女孩在笑——为什么要看？ |
| **标题短 + 大** | 封面上的文字 ≤ 8 个字，字体加粗加描边 | "AI 生成的我，妈妈认不出" | "使用人工智能技术生成的一个关于亲情的短片作品" |
| **对比度要高** | 小图浏览时，低对比度的封面是一片灰 | 亮暗分明，色彩饱和 | 整体灰蒙蒙，人和背景融在一起 |
| **色调统一** | 封面色调 = 短片整体风格色调 | 悬疑片用冷蓝色调封面 | 橙色暖调封面 + 黑白正片 |

### 封面制作工具

- **剪映**：直接提取视频中的关键帧，加文字模板——最方便
- **醒图 / 美图秀秀**：如果不满意提取的帧，可以专门做一张封面图
- **Canva**：模板丰富，适合设计感较强的封面

## 📊 好标题 vs 坏标题

| 维度 | 好标题 | 坏标题 | 为什么 |
|:--|:--|:--|:--|
| **AI 短片类** | "我用 AI 拍了部电影，我妈哭了" | "AI 生成短片作品" | 好标题有结果（妈哭了），有悬念（为什么哭）；坏标题只是描述 |
| **情感类** | "分手第 30 天，AI 帮我续写了结局" | "一个关于分手的故事" | 好标题有具体数字+AI介入+反转期待；坏标题太泛 |
| **悬疑类** | "这部 AI 短片，最后一秒颠覆一切" | "AI 悬疑短片" | 好标题制造好奇心缺口；坏标题什么都没说 |
| **教程类** | "0 基础 3 天做出 AI 短片，我的 8 个步骤" | "AI 短片制作教程" | 好标题有受众+时间+方法论；坏标题没有任何差异化信息 |
| **治愈类** | "加班到凌晨，AI 给我做了个动画哄睡" | "治愈系 AI 动画" | 好标题有具体场景和情绪共鸣；坏标题只有笼统风格 |

### 标题公式

```
[受众痛点/共鸣场景] + [AI 做了什么] + [出人意料的结果/数字/反转]
```
