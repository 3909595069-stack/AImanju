# 🤝 人机协作工作流

> AI不是替你写，是帮你写。把剧本从头到尾全丢给AI = 得到一坨废话。

## 七步协作流程

| 步骤 | 谁做 | 做什么 |
|------|------|--------|
| 1. 竞品研究 | 你+AI | 用AI分析最近30天最火的AI漫剧共同规律 |
| 2. 创意头脑风暴 | AI | 让AI生成10个不同角度的故事大纲，你选最好的 |
| 3. 写钩子 | 👤 你 | 亲手写出前3秒的钩子画面+台词——这是AI最弱的地方 |
| 4. 剧本初稿 | AI | 用五步提示词框架，让AI基于你的钩子扩展完整剧本 |
| 5. 精修 | 👤 你 | 读一遍，删"AI味"句子改成口语，调整节奏 |
| 6. 分镜转换 | AI | 把精修后的剧本发给AI拆成15-25个分镜 |
| 7. 分镜检查 | 👤 你 | 检查节奏感——景别搭配是否单调、有没有全中景的问题 |

## 核心原则

- **钩子必须人写** — AI趋向平均，写不出尖锐的开头
- **你说了算** — AI是建议机器，最终决定权永远在你手上
- **边做边改** — 不要期待一步到位，视频生成后才发现问题回来改剧本很正常
