# 🤖 五步提示词框架

> 大多数人直接让AI"写一个剧本"，但不教它"什么是好剧本"。这个框架是你需要教给AI的所有东西。

## 完整五步提示词

```markdown
【第一步：角色定义】
你是一位专业短视频编剧，擅长打造高留存率的开篇钩子、设计紧凑的冲突升级、
安排意想不到的情节反转、创作自然的口语化对白。

【第二步：核心信息】
- 题材：[重生逆袭 / 玄幻修仙 / 都市甜宠 / 末世生存 / 霸总]
- 时长：约60-90秒
- 平台：抖音（竖屏9:16）
- 目标观众：[例如：18-35岁女性]
- 核心情感基调：[爽 / 虐 / 甜 / 燃 / 悬疑]

【第三步：结构要求】
请严格按照以下结构生成剧本，每个节拍标注时间码：
1. 钩子 (0-5s)：模式中断型开头，用出人意料的画面+一句话抓住观众
2. 建置 (5-15s)：快速揭示主角处境+今天的"反常"
3. 冲突升级 (15-45s)：障碍从弱到强，主角一次次被逼退
4. 反转/高潮 (45-70s)：颠覆观众认知的一刻，情绪爆发
5. 留白结尾 (70-90s)：用一句话/一个画面留悬念，引导"下集"

要求每30秒至少有一个情绪转折点。

【第四步：风格约束】
- 语言必须口语化，每句对白不超过15个字
- 不要出现"AI腔"：比如"让我们回顾""标志性的是""在这个过程中"等
- 对白推动剧情或揭示性格，不做单纯说明
- 参考语气：朋友之间说悄悄话的感觉

【第五步：输出格式】
请以标准剧本格式输出，包含：
- 每场戏的场景标题（地点/时间）
- 角色动作描述（具体的外部动作，不能是心里描写）
- 对白
- [画外音/旁白标注]
- [关键镜头建议（景别+运镜）]
- [音效/氛围提示]
- 每个角色的人物设定描述（外貌、服装、性格特征）
```
