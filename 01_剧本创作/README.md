# 01 · 剧本创作 — 先把故事写出来

> 没有剧本，后面全是瞎搞。剧本决定了你需要什么角色、什么场景、多少个镜头。这是整个AI漫剧的"地基"。

## 🔍 这一步在做什么？

写出短剧的**故事内容**——谁在什么情况下做了什么，发生了什么转折。

- 一集 AI 漫剧通常：**1-2 分钟**
- 大约字数：**200-400 字**
- 核心要求：在 60-90 秒内抓住观众、带他们经历情绪过山车、留下谜团

## 💡 为什么先做这一步？

剧本就像盖房子的图纸。一部 AI 漫剧能不能火，**80% 取决于剧本质量**——再精美的画面也救不了一个无聊的故事。

### 选题验证三步法

1. 去抖音/小红书搜"AI漫剧"，按"最热"排序，看近30天爆款
2. 把爆款的题材、人设、结构列出来，找共同点
3. 在选定题材里，加一个**独特的细节**做差异化

## 📖 叙事结构 — 好故事是有公式的

### 结构A：微戏剧弧线 (30-60秒超短剧)

| 阶段 | 时间 | 功能 |
|------|------|------|
| **钩子 Hook** | 0–3s | 瞬间抓住注意力 |
| **铺垫 Build-up** | 3–8s | 快速呈现冲突 |
| **高潮 Climax** | 8–12s | 情感/视觉冲击最强 |
| **收尾 Resolution** | 12–15s | 快速收束，留悬念 |

### 结构B：好莱坞三拍 (Hook → Build → Payoff)

| 节拍 | 在做什么 | 常见错误 |
|------|---------|---------|
| **Hook** | 告诉观众"为什么值得看" | 做自我介绍——观众不想知道你是谁 |
| **Build** | 一次只发展一个想法，每30秒一个转折 | 信息过载 |
| **Payoff** | 兑现Hook中的承诺 | 结尾太模糊，观众觉得"就这？" |

### 结构C：经典三幕剧 (1-2分钟短剧)

| 幕 | 功能 | 时间分配 |
|----|------|---------|
| **第一幕·建置** | 设定世界、引入主角、建立冲突 | 前15秒 |
| **第二幕·对抗** | 障碍升级、转折、情感加深 | 中间30-40秒 |
| **第三幕·解决** | 高潮对决、人物成长、情感回报 | 最后15-20秒 |

### 黄金结构口诀

> 钩子开头(0-3s) → 冲突升级(3-30s) → 反转高潮(30-50s) → 留白结尾(50-60s)
> **每30秒必须有一个情绪转折**

## 🪝 钩子设计 — 前3秒定生死

抖音/快手用户平均只给你 **0.5-1秒** 来决定是否停下。

### 五种钩子策略

| 策略 | 原理 | AI漫剧示例 |
|------|------|-----------|
| 🎯 模式中断 | 出人意料的画面/声音 | 婚礼新娘突然消失，只剩婚纱 |
| ❓ 问题钩子 | 抛出观众关心的问题 | "如果你重生回高考当天但三年没碰书……" |
| 👀 结果预告 | 先展示惊人结果 | 废墟中唯一活人，切回"三天前" |
| 💥 大胆断言 | 反常识的声明 | "所有人都以为魔王是坏人，但他才是……" |
| 📖 故事开场 | 直接在冲突中开始 | "分手吧。"茶杯摔碎，笑容僵住。 |

> ⚠️ **钩子最好由你亲手写！** AI天生趋向"平均"，写不出尖锐的钩子。

## ⚔️ 冲突设计 — 没有冲突就没有故事

### 四种冲突类型

| 类型 | 解释 | AI漫剧场景 |
|------|------|-----------|
| 角色 vs 角色 | 目标对立 | 男女主误会、正反派斗智 |
| 角色 vs 环境 | 外部障碍 | 末日生存、天灾 |
| 角色 vs 自身 | 内心挣扎 | 复仇还是宽恕 |
| 角色 vs 社会 | 对抗大系统 | 打破规则、挑战权威 |

### 冲突升级五步原则

1. 明确主角目标 → 创建障碍
2. 构建对抗力量 → 让阻碍越来越强
3. 设置转折点 → 以为解决了，发现更大的问题
4. 情感冲突 + 利益冲突 + 价值观冲突叠加
5. 每一拍都比前一拍更紧张——**永远不要让观众松口气**

## 🔄 反转设计 — "卧槽"的一刻

| 反转类型 | 怎么做 | AI漫剧示例 |
|---------|--------|-----------|
| **身份反转** | 真实身份揭露 | 温暖前辈其实是幕后反派 |
| **真相反转** | 以为的真相被颠覆 | 以为被抛弃，其实对方暗中保护十年 |
| **动机反转** | 真正原因被揭露 | 霸道老板其实在培养接班人 |
| **处境反转** | 权力关系逆转 | 反派跪下求饶，弱者变强者 |
| **因果反转** | 恶作剧反弹 | 下毒的人自己中毒 |

> 🎯 **反转原则：** ① 前文必须有伏笔 ② 应改变观众对整个故事的理解 ③ 每个剧本至少1-2个关键反转点
