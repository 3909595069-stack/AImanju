# 🗣️ 如何避免"AI味"

> AI生成的剧本有一个致命问题——读起来不像人说的话。以下是具体的改写技巧。

## Before / After 对比

| ❌ AI味的写法 | ✅ 人味的改法 |
|-------------|-------------|
| "在这个关键时刻，我做出了一个改变我一生的决定。" | "就现在。赌一把。" |
| "他的眼神中透露出一丝不易察觉的忧伤。" | "他别过脸去。" |
| "经过这段时间的相处，我们的关系发生了质的变化。" | "他给我倒了杯水。第一次。" |
| "这是一个标志性的时刻，预示着我们团队的新篇章。" | "从今天起，这里我说了算。" |
| "让我们回顾一下过去发生的一切……" | "你还记得那天吗？" |
| "或许我们可以考虑一下其他方案？" | "换个方式。" |
| "此时此刻，我的内心充满了感激之情。" | "谢了。" |

## 五个改写技巧

1. **每句对白不超过15个字** — 超过就拆成两句，或者删掉废话
2. **把「形容词」换成「动作」** — 不写"他很感动"，写"他别过脸，抬手揉了揉眼睛"
3. **删掉"AI味"信号词** — "标志性""里程碑""过程中""让我们回顾""深刻意识到"
4. **把书面语改成口语** — "为何"→"为什么"，"或许"→"也许"，"如此"→"这么"，"此刻"→"现在"
5. **大声读一遍** — 读不出来/读得别扭/不像正常人说话 → 改写

## "AI味"高频词黑名单

标志性、里程碑、过程中、让我们回顾、深刻意识到了、发生了质的变化、经过这段时间的相处、此时此刻、充满了感激之情、不易察觉的、改变一生的、预示着的
