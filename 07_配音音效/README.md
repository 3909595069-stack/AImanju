# 配音音效：让短剧有声音

> 画面只是剧的一半——没有声音的短片是半成品。好的音频能把60分的画面提到80分，差音频能把90分的画面毁到50分。

## 🔍 这是啥？

**配音音效**，在 AI 短片工作流中包括三个子环节：

| 子环节 | 内容 | 优先级 |
|:--|:--|:--|
| **配音** | 角色对白/旁白/独白的人工智能语音生成 | ⭐⭐⭐ 必须 |
| **音效 (SFX)** | 环境音（雨声、风声、街道）、动作音（脚步、开门、物品碰撞） | ⭐⭐ 重要 |
| **BGM** | 背景音乐，奠定情绪基调和节奏 | ⭐⭐ 重要 |

> 三条线分别制作，在 [[../08_剪辑合成/README|剪辑合成]] 环节混到一起。

## 💡 为啥做？

1. **观众对音频的容忍度远低于画面**：画面稍微差一点，观众可能接受。但配音像机器人、BGM不搭、音画不同步——观众3秒划走
2. **音频是情绪放大器**：同一个画面配上不同的BGM，情绪完全不同——恐怖→温情→搞笑，音频说了算
3. **弥补AI画面的不足**：AI生成的画面可能在某些细节上不够精准，但好的配音和音效会引导观众的注意力，让他们"看"到你想让他们看的东西
4. **声音是沉浸感的锚点**：环境音（雨声/风声/街道嘈杂）让观众相信"这个场景是真实的"——这是AI画面难以单独完成的事

## 📝 配音脚本工程

### 配音脚本标注体系（Audio Tags）

2025年主流TTS工具支持的**内联情感标签**——写在配音脚本里，AI就能理解：

| 类别 | 标签示例 | 效果 |
|:--|:--|:--|
| 😊 **情绪** | `[excited]` `[annoyed]` `[flustered]` `[sarcastically]` `[casual]` | 控制角色情绪基调 |
| ⏱️ **节奏** | `[fast-paced]` `[hesitates]` `[pause]` `[drawn out]` | 控制语速和停顿 |
| 🗣️ **声音特征** | `[childlike tone]` `[deep voice]` `[robotic tone]` `[whispers]` | 切换声音属性 |
| 💬 **副语言** | `[laughs]` `[sighing]` `[panicking]` `[crying]` | 笑声、叹气、恐慌等非语言表达 |
| 🔀 **对话互动** | `[interrupting]` `[overlapping]` `[cuts in]` | 控制对话打断与重叠 |

**标注脚本示例**：
```
角色A（女主）：[casual] 今天天气真不错啊。
角色B（男主）：[interrupting] 等等！你看到了吗？
角色A（女主）：[panicking] 什么东西？！刚才窗户那边...
角色B（男主）：[whispers, drawn out] 别出声。慢慢往后退。
角色A（女主）：[sighing, relieved] 原来只是只猫...
```

### 中文配音脚本优化技巧

| 问题 | 原因 | 解决方案 |
|:--|:--|:--|
| 声音太平，没感情 | AI默认用中性语气 | 在文本中标注情绪提示：`[悲伤]` `[激动]` `[犹豫]` |
| 断句奇怪，像在读 | AI不理解语义断句 | 用标点精确控制：句号=长停，逗号=短停；长句拆成短句 |
| 语速不对 | 默认语速与场景不匹配 | 紧张对话 1.1-1.2x，日常对话 1.0x，抒情旁白 0.85-0.95x |
| 多角色区分度不够 | 选了相似的音色 | 选音色时确保音高差异明显（一偏高一偏低）；后期EQ微调 |
| 中文多音字读错 | AI分不清多音字 | 用拼音标注：`银行(háng)` vs `行(xíng)走`；或换同义词 |
| 数字/英文混读怪异 | 中英文混排打乱AI节奏 | 中文数字写汉字（"一百二十三"而非"123"）；英文单词前后加空格 |
| 对话过于生硬 | 缺乏口语化表达 | 加入语气词和口语化改写："你在干什么？"→"你干嘛呢？" |

### 多角色配音工作流

```
1. 建立角色音色对照表 → 每个角色绑定固定音色ID
2. 按角色分段导出配音文本 → 每个角色一段
3. 逐个角色生成AI语音 → 保持同一工具/同一版本
4. 后期微调EQ → 不同角色音轨做频率差异化
5. 导入剪辑软件 → 逐角色对齐口型和时间轴
```

### BGM选用原则

1. **纯器乐优先**：避免带歌词的歌曲——歌词会和人声对白打架。人声+BGM唱歌=观众大脑过载
2. **情绪先行，好听在后**：先确定场景的情绪（温馨/紧张/悲伤/振奋），再按情绪找BGM——而非按"好听"
3. **循环点检查**：找有自然循环点的BGM，方便拉长匹配视频长度。如果BGM循环有明显"接缝"，换一首
4. **版权意识**：发布到抖音/B站/小红书前，确认BGM版权允许商业使用。CC0协议最安全
5. **BGM留白**：不要把BGM铺满全片。关键时刻（情绪爆发/重大揭示）关掉BGM——安静本身就是最强的"音效"
6. **避免情绪BGM绑架**：BGM是陪衬，不是主角。如果观众说"这首歌真好听"而不是"这个故事真感人"，你就失败了

## 🎚️ 混音工程（Audio Mixing）

### 分层音量公式（参考起点）

在 [[../08_剪辑合成/README|剪辑合成]] 中调整各音轨的相对音量：

| 音轨类型 | 建议音量 (相对0dB) | 说明 |
|:--|:--|:--|
| **人声（对白/旁白）** | **0 dB** | 基准音量，所有其他音轨相对它来调 |
| **重点音效（门声/碰撞/枪声）** | **-3 ~ -6 dB** | 比人声稍低，但要能被注意到 |
| **BGM（高潮段落）** | **-12 ~ -15 dB** | 人声说话时BGM要明显退后 |
| **BGM（过渡/气氛段）** | **-15 ~ -18 dB** | 安静段落BGM可以稍大一点 |
| **环境音** | **-12 ~ -18 dB** | "能感觉到但不需要刻意听" |
| **微妙音效（脚步/衣物摩擦）** | **-18 ~ -24 dB** | 增加真实感但不应抢夺注意力 |

### 音频闪避（Ducking）—— 最重要的自动混音技巧

**原理**：当检测到人声时，BGM自动降低音量；人声停止后BGM自动恢复。

| 工具 | 操作 |
|:--|:--|
| **剪映** | 选中音乐音轨 → "音频闪避" → 选择人声音轨作为触发源 → 调整闪避深度和过渡时长 |
| **Premiere Pro / Audition** | 将对话音轨设为"闪避触发源" → 调整Threshold/Reduction → 调Attack(~10ms)和Release(~100ms) |

### EQ均衡——让声音更清晰

- **高通滤波（HPF）**：人声音轨切掉80-120Hz以下低频 → 消除嗡嗡声，对话瞬间"清爽"
- **人声清晰度提升**：在 **2-4kHz** 频段稍微提升 → 人声"存在感"和"清晰度"的关键区域
- **为人声让路**：音乐轨略微衰减150-300Hz → 避免和对话中低频"打架"

### 完整混音工序（7步法）

| 步骤 | 操作 | 说明 |
|:--|:--|:--|
| **1. 降噪** | 对所有人声音轨做降噪处理 | 先降噪再混音——否则压缩后噪声会被放大 |
| **2. 均衡 (EQ)** | 人声HPF+清晰度提升 / 音乐为人声让路 | 让人声和音乐在频谱上"分开" |
| **3. 压缩** | 人声Ratio 2:1~4:1，Attack 5-10ms | 让轻柔处更响，响亮处不爆音 |
| **4. 闪避 (Ducking)** | BGM对人声做自动闪避 | 这是最关键的一步 |
| **5. 混响** | 人声加微量混响（Room/Plate） | 不要干声——太"死"；不要太多——太"远" |
| **6. 音量平衡** | 对照分层音量公式逐轨调整 | 人声始终最清晰 |
| **7. 母带输出** | 限幅器输出上限 -1dB，响度 -16 LUFS | 防止爆音，统一终响亮 |

### 混音检查清单

- [ ] 人声清晰可辨，无背景噪音
- [ ] BGM在人声进入时自动降低（Ducking已开启）
- [ ] 音效与画面动作的时间差 < 0.1秒
- [ ] 转场时有自然的音频过渡（淡入淡出），不要硬切音频
- [ ] 全片音量一致，不需要观众频繁调节音量
- [ ] **耳机和手机外放各听一遍**——两种环境表现差异很大
- [ ] 不同的BGM之间做了交叉淡入淡出（crossfade），无突然切换感
- [ ] 爆音检测：全片无超过0dB的削波失真

## 🎬 配音实操流程

### 从分镜脚本到配音成品

```
步骤1：准备配音脚本
  ├── 从分镜脚本中提取所有对白
  ├── 添加情绪标签和停顿标注
  ├── 建立角色-音色对照表
  └── 按角色切分配音段落

步骤2：选择工具和音色
  ├── 新手 → 剪映内置（最快）
  ├── 进阶 → MiniMax（中文情感最优）
  └── 专业 → MiniMax + ElevenLabs 混用

步骤3：逐角色生成
  ├── 同一角色的所有台词用同一音色ID
  ├── 调整语速参数（紧张1.1x / 日常1.0x / 抒情0.9x）
  ├── 导出每个角色的音频文件
  └── 文件命名：角色名_场景号_WAV格式

步骤4：粗对时间轴
  ├── 在剪辑软件中将配音拖入时间轴
  ├── 按分镜顺序排列
  └── 微调每段配音的起止位置

步骤5：精调
  ├── 对话间加入自然停顿（0.2-0.5秒）
  ├── 情绪高潮处加0.3秒额外停顿
  └── 多角色对话场景检查"抢话"和"留白"
```

### AI配音自然度提升银弹技巧

| 技巧 | 操作 |
|:--|:--|
| **参考音频对比法** | 录一遍你自己读的参考音频（手机即可），和AI配音对比着听——差在哪立刻就知道 |
| **标点控制节奏** | 句号=长停、逗号=短停、省略号=渐弱、破折号=语气转折 |
| **口语化改写** | "你是否愿意与我共进晚餐？"→"一起吃个饭吧？" |
| **语速分段差异化** | 同一角色日常对话1.0x、紧张对话1.15x、内心独白0.9x |
| **EQ微调角色化** | 年轻角色→略提高频(5k+) / 年长角色→略提低频(150-300Hz) / 反派→加轻微失真 |

## 🐛 AI配音常见问题与排错

| # | 问题 | 根因 | 解决方案 |
|:--|:--|:--|:--|
| 1 | 🤖 **像机器人** | AI默认中性语气 | 标注情绪标签 + 口语化改写 + 调整语速 |
| 2 | 📖 **断句奇怪** | AI不理解语义 | 用标点精确控制停顿，长句拆短句 |
| 3 | ⚡ **语速不对** | 默认语速与场景不匹配 | 紧张1.1-1.2x / 日常1.0x / 抒情0.85-0.95x |
| 4 | 👥 **多角色分不清** | 选了相似音色 | 确保音高差异明显 + EQ微调差异化 |
| 5 | 🔤 **多音字读错** | AI不认识多音字 | 拼音标注：`银行(háng)` vs `行(xíng)走` |
| 6 | 🔢 **数字/英文混乱** | 中英文混排打乱节奏 | 中文数字写汉字、英文词前后加空格 |
| 7 | 📢 **爆音/削波** | 音量超出0dB上限 | 生成时降低输出音量，后期限幅器-1dB |
| 8 | 🎭 **两个角色声音串了** | 文件管理混了 | 严格按角色-场景号命名，文件头写入角色名 |
