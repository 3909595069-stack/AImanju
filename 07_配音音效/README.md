# 配音音效：让短剧有声音

> 画面只是剧的一半——没有声音的短片是半成品。好的音频能把 60 分的画面提到 80 分。

## 🔍 这是啥？

**配音音效**，在 AI 短片工作流中包括三个子环节：

| 子环节 | 内容 | 优先级 |
|:--|:--|:--|
| **配音** | 角色对白/旁白/独白的人工智能语音生成 | ⭐⭐⭐ 必须 |
| **音效 (SFX)** | 环境音（雨声、风声、街道）、动作音（脚步、开门、物品碰撞） | ⭐⭐ 重要 |
| **BGM** | 背景音乐，奠定情绪基调和节奏 | ⭐⭐ 重要 |

> 三条线分别制作，在 [[../08_剪辑合成/README|剪辑合成]] 环节混到一起。

## 💡 为啥做？

1. **观众对音频的容忍度远低于画面**：画面稍微差一点，观众可能接受。但配音像机器人、BGM 不搭、音画不同步——观众 3 秒划走
2. **音频是情绪放大器**：同一个画面配上不同的 BGM，情绪完全不同——恐怖→温情→搞笑，音频说了算
3. **弥补 AI 画面的不足**：AI 生成的画面可能在某些细节上不够精准，但好的配音和音效会引导观众的注意力，让他们"看"到你想让他们看的东西

## 📊 混音公式

在 [[../08_剪辑合成/README|剪辑合成]] 中调整各音轨的相对音量。以下数值为参考起点，根据具体素材微调。

| 音轨类型 | 建议音量 (相对 0dB) | 说明 |
|:--|:--|:--|
| **人声（对白/旁白）** | **0 dB** | 基准音量。所有其他音轨相对于它来调 |
| **BGM（高潮段落）** | **-12 ~ -15 dB** | 人声说话时 BGM 要明显退后 |
| **BGM（过度/气氛段）** | **-15 ~ -18 dB** | 安静段落 BGM 可以稍大一点，但不能压过环境音 |
| **环境音** | **-12 ~ -18 dB** | 环境音（雨/风/街道）保持在"能感觉到但不需要刻意听"的水平 |
| **重点音效（门声/碰撞/枪声）** | **-3 ~ -6 dB** | 比人声稍低，但要能被注意到 |
| **微妙音效（脚步/衣物摩擦）** | **-18 ~ -24 dB** | 增加真实感但不应抢夺注意力 |

### 混音检查清单

- [ ] 人声清晰可辨，无背景噪音
- [ ] BGM 在人声进入时自动降低（Ducking 效果）
- [ ] 音效与画面动作的时间差 < 0.1 秒
- [ ] 转场时有自然的音频过渡（淡入淡出），不要硬切
- [ ] 全片音量一致，不需要观众频繁调节音量
- [ ] 用耳机和手机外放各听一遍——两种环境表现差异很大

### BGM 选用原则

1. **纯器乐优先**：避免带歌词的歌曲——歌词会和人声对白打架
2. **情绪匹配**：先确定场景的情绪（温馨/紧张/悲伤/振奋），再按情绪找 BGM，而非按"好听"找
3. **循环点检查**：找有自然循环点的 BGM，方便拉长到匹配视频长度
4. **版权意识**：发布到抖音/B站/小红书前，确认 BGM 的版权允许商业使用（如果你是商业发布）

## 🐛 AI 配音像机器人的解决方案

| 问题 | 原因 | 解决方案 |
|:--|:--|:--|
| 声音太平，没感情 | AI 默认用中性语气 | 在配音文本中标注情绪提示：`[悲伤]` `[激动]` `[犹豫]`，部分工具会识别 |
| 断句奇怪，像在读 | AI 不理解语义，不知道在哪断 | 用标点符号精确控制停顿：句号=长停，逗号=短停。必要时把长句拆成短句 |
| 语速不对 | 默认语速与场景不匹配 | 在工具中调整语速参数：紧张对话 1.1-1.2x，抒情旁白 0.85-0.95x |
| 多角色声音区分度不够 | 选了相似的音色 | 选音色时确保音高差异明显（一个偏高一个偏低）；或在后期用 EQ 微调 |
| 中文多音字读错 | AI 分不清多音字 | 用拼音标注：`银行(háng)` vs `行(xíng)走`；或换一个同义词 |
| 数字/英文混读怪异 | 中英文混排打乱 AI 节奏 | 中文数字写成汉字（"一百二十三"而非"123"）；英文单词前后加空格 |

> **银弹技巧**：录一遍你自己读的参考音频（用手机就行），然后把这个音频和 AI 配音对比着听。AI 差在哪，你一听就知道——下次调整参数就有方向。
