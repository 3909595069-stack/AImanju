# 剪辑合成：把碎片拼成一部剧

> 你手里有一堆视频片段、配音文件、BGM、音效——现在把它们变成一部能让陌生人看完的作品。

## 🔍 这是啥？

**剪辑合成**，是将前面所有工序的产出——[[../06_图生视频/README|视频片段]]、[[../07_配音音效/README|配音音频]]、BGM、音效——在时间轴上组装为一部完整短片的环节。

这一步的难点不在技术，在你的审美判断。**剪辑 = 选择 + 节奏**。

## 💡 为啥做？

1. **拼在一起 ≠ 一部剧**：把 20 个片段按顺序排好，那不叫剪辑，那叫"排队"。剪辑需要剪掉多余的、拉长需要的、重新排列顺序。
2. **节奏是短片的生命**：一样的素材，节奏对了是"好看"，节奏不对是"好困"。节奏由你来定——工具只是执行。
3. **这是最后一道质量关口**：在剪辑台上，你会发现分镜阶段没想到的问题——两个镜头接不上、某个镜头太长了、情绪断档了。现在修复比发布后删视频代价小得多。

## 🛠️ 剪映 9 步流程

以下以**剪映**（最推荐的 AI 短片剪辑工具）为例：

| 步骤 | 操作 | 要点 |
|:--|:--|:--|
| **1. 新建项目** | 剪映 → 开始创作 → 选择 16:9（横屏）或 9:16（竖屏） | 横屏更适合叙事短片，竖屏适合抖音/小红书 |
| **2. 导入素材** | 将所有视频片段拖入素材库 | 按镜号重命名：`S01_全景便利店` `S02_主角进门`，方便查找 |
| **3. 粗剪** | 按分镜顺序将视频放入时间轴，不调时长，只排顺序 | 这一步只关心"顺序对不对"，不关心"好不好看" |
| **4. 精剪** | 调整每个片段的起止点，删掉多余的帧 | 使用下面的"节奏控制表"确定每个镜头留多长 |
| **5. 转场** | 在片段之间添加转场效果 | 见下方"转场原则" |
| **6. 配音轨** | 导入 [[../07_配音音效/README|配音音频]]，对齐到对应画面 | 关键：口型/动作和声音的时间差 < 0.1 秒 |
| **7. 加 BGM** | 拖入背景音乐轨，调整音量，设置淡入淡出 | 参考混合公式设置音量 |
| **8. 加音效** | 在对应位置添加音效（脚步声、开门声、环境音等） | 不要贪多——只加真正需要的音效 |
| **9. 调色 + 字幕 + 导出** | 统一调色 → 添加字幕 → 导出 | 见下方调色参数表和导出规格 |

## 📊 节奏控制表

决定一个画面该停留多久——这可能是剪辑中最被低估的技能。

| 内容类型 | 建议时长 | 为什么 | 例子 |
|:--|:--|:--|:--|
| **紧张/动作** | **1.5 - 2 秒** | 短促 = 紧张感。观众来不及"看够"就被推到下一个画面 | 追逐、打斗、发现危机、快速反应 |
| **对话** | **2 - 3 秒** | 在说完一句对白后多留 0.3-0.5 秒。让观众有时间"接收"信息 | 两人对话、角色独白、关键信息传递 |
| **日常动作** | **2 - 4 秒** | 完整的微动作需要这个时长来"完成" | 倒水、走路、坐下、看书 |
| **抒情/空镜** | **4 - 6 秒** | 给观众呼吸的时间。抒情段落靠"长"来积累情绪 | 风景空镜、雨中街道、夕阳、角色沉思 |
| **特写（情绪）** | **2 - 3 秒** | 特写的情绪密度高，不需要太长——过长反而尴尬 | 眼神变化、手部颤抖、嘴角微笑 |
| **建立镜头** | **3 - 5 秒** | 开场建立镜头需要足够长，让观众建立起空间认知 | 新场景的第一个全景/远景 |

> **自查方法**：全片剪完后，闭眼听一遍——不看画面，只感受节奏。如果某一段让你想"快进"，那段就有节奏问题。

## 🎨 调色参数建议

这是**起点参数**，不是最终数值——每部短片都需要根据素材微调。但有一个起点比从零开始快得多。

| 参数 | 建议范围 | 效果 | 适用场景 |
|:--|:--|:--|:--|
| **色温** | ±0 ~ +10（偏暖）<br>0 ~ -10（偏冷） | 暖 = 温馨/怀旧<br>冷 = 冷峻/科技/悲伤 | 生活剧 +5~+8<br>悬疑/科幻 -5~-8 |
| **饱和度** | -5 ~ +10 | 高饱和 = 活力/动漫感<br>低饱和 = 克制/文艺/悬疑 | 日系 +8~+10<br>文艺片 -5~-10<br>生活剧 ±0 |
| **对比度** | +5 ~ +15 | 高对比 = 电影感/戏剧性<br>低对比 = 柔和/梦幻 | 写实/电影 +10~+15<br>温馨/治愈 ±0~+5 |
| **锐化** | +10 ~ +25 | 增强细节清晰度 | AI 生成画面通常建议 +15~+20，因为 AI 素材偏软 |
| **暗角** | +5 ~ +15 | 四角变暗，聚焦画面中心 | 情绪特写 +10~+15<br>全景/远景 +0~+5 |
| **褪色（Fade）** | +0 ~ +10 | 黑色不纯黑，复古胶片感 | 文艺/怀旧 +5~+10<br>商业化/明亮 ±0 |

### 调色黄金规则

1. **一部片，一套色**：所有镜头用同一套调色参数。不要一个镜头暖一个镜头冷
2. **先调色温再调其他**：色温决定了整个基调，定下来之后再动其他参数
3. **用 LUT 快速统一风格**：剪映内置了电影感 LUT，找一个适合你的片风，一键应用——比自己摸索快很多
4. **手机屏幕检查**：在电脑上调好色，导出到手机上再看一遍。90% 的观众在手机上看你的作品
5. **少即是多**：新手最容易犯的错——每个参数都拉到极限。"稍微调一点"通常比"拉满"效果好

## ✂️ 转场原则

| 原则 | 说明 |
|:--|:--|
| **大多数情况：硬切** | 90% 的转场应该是硬切（直接换画面，无特效）。专业电影 95% 都是硬切 |
| **需要暗示时间/空间变化时：淡入淡出** | 黑场淡出→淡入 = "一段时间过去了" 或 "换了一个地方" |
| **需要暗示关联时：叠化** | 两个画面短暂重叠 = "这两件事同时发生"或"这就是他脑中的画面" |
| **永远不要用花哨转场** | 翻页、百叶窗、3D 翻转、星形擦除——这些都是"廉价的炫技"，让作品一秒变业余 |
| **转场时长** | 硬切 = 0 帧；淡入淡出 = 0.5-1 秒；叠化 = 0.3-0.5 秒 |

> 一句话：如果观众注意到了你的转场方式，你已经失败了。好的转场是**看不见的**。

## 📦 导出规格

| 平台 | 分辨率 | 帧率 | 格式 | 码率建议 |
|:--|:--|:--|:--|:--|
| **抖音** | 1080×1920 (9:16竖屏) | 30fps | MP4 H.264 | 6-10 Mbps |
| **B站** | 1920×1080 (16:9横屏) | 24/30fps | MP4 H.264 | 10-15 Mbps |
| **小红书** | 1080×1440 (3:4竖屏) 或 1920×1080 | 30fps | MP4 H.264 | 6-8 Mbps |
| **快手** | 1080×1920 (9:16竖屏) | 30fps | MP4 H.264 | 6-10 Mbps |
| **微信视频号** | 1080×1920 (竖屏) 或 1920×1080 (横屏) | 30fps | MP4 H.264 | 6-10 Mbps |
| **通用存档** | 1920×1080 | 24fps | MP4 H.264 | 15-20 Mbps |

> 24fps = 电影感；30fps = 电视感。叙事短片建议 24fps。

## 🐛 常见问题

| 问题 | 解决方案 |
|:--|:--|
| 片段之间跳帧 | 在剪映中把交接处的片段各延长 0.1 秒做微叠化 |
| 配音和画面对不上 | 先把配音轨放好，再根据配音节奏调整画面时长——让画面服务音频 |
| 导出的视频画质变差 | 检查导出码率是否过低；原片是否经过多次压缩；导出前关闭剪映的"智能画质优化" |
| BGM 压过人声 | 在剪映中使用音频 Ducking（闪避）功能——检测到人声时自动降低 BGM |
| 全片太长或太短 | 新手短片建议 1-3 分钟。超过 5 分钟需要非常强的节奏控制力 |
