# 实景素材拍摄清单（精简版）

> 用途：第1集《暴雨夜》AI生图场景参考图  
> 总量：约80张 | 次数：2次 | 总耗时：约2.5小时  
> 设备：荣耀400手机 | 时机：台风天优先拍夜景

## 空间布局速查

```
                        天心湖
                      ~~~~~~~~~~
        一鉴桥(平直石桥)          中流桥(平直石桥)
            |                        |
          [30m]                   [直走]
            |                        |
        校训石(拐角处)               |
            |                        |
       [左拐100m]                   |
            |                        |
    瑞樟路←──图书馆(临湖)──→自然区方向
    (左转)    (中轴)    (右转)
                        →朱子学院→荷花池→茶山
```

**林夕第1集路线**：图书馆出来→左转瑞樟路→走100米→左拐→校训石→前方一鉴桥30米  
**鬼打墙逻辑**：三条路（中流桥/瑞樟路/自然区）出发→全部强制回到校训石

## 第一次拍摄：夜景专场（台风还在的晚上）

> 耗时约1.5小时 | 目标约50张  
> 先趁黄昏微光拍空间关系，天黑后拍地标夜景

### 一、空间关系（趁还有光，先拍！）约25张

| # | 站哪 | 拍什么 | 张数 |
|---|------|--------|------|
| 1 | 图书馆门口台阶正中 | 正前方→中流桥方向（林夕出门第一眼） | 2 |
| 2 | 图书馆门口台阶 | 左边→瑞樟路方向 | 2 |
| 3 | 图书馆门口台阶 | 右边→自然区方向 | 2 |
| 4 | 沿瑞樟路走100米到拐角 | 拐过去第一眼看到校训石的角度 | 3 |
| 5 | 站校训石旁 | 360°环拍：一鉴桥/图书馆来路/瑞樟路来路/天心湖 | 8 |
| 6 | 站三岔路口中央 | 360°环拍：中流桥/瑞樟路/自然区/来路 | 8 |

> 空间关系一张都不能省——这是所有AI生图的一致性地基。

### 二、地标夜景 约20张

| 地标 | 拍什么 | 张数 |
|------|--------|------|
| 校训石 | 路灯昏黄光照在刻字上（和第1集镜头04同款光线） | 2 |
| 图书馆 | 门口台阶夜景+门口灯光 | 2 |
| 中流桥 | 桥面夜景+路灯（桥面石板反光） | 3 |
| 天心湖 | 湖面路灯倒影（金色光带） | 2 |
| 一鉴桥 | 夜景一张 | 1 |
| 路灯 | 灯柱样式+光色+潮湿地面反光 | 3 |
| 雨天氛围 | "下雨的校园路"的感觉——不拍特定地标，拍氛围 | 4 |
| 雨点湖面 | 雨点砸在天心湖水面的涟漪 | 3 |

### 三、夜景拍摄要点

- **打伞护手机**：站好位置→伞挡雨→取出手机→快拍→收回。每张多拍1张备份防雨滴糊镜
- **优先拍有遮挡的地方**：图书馆门口台阶下、中流桥桥头路灯旁
- **露天场景**：等雨小冲出去快拍
- **荣耀400夜景模式**：手靠栏杆/墙当三脚架，曝光补偿-0.7防灯光过曝
- **地面反光是核心资产**：潮湿路面+路灯反光 → AI做新海诚光影的绝佳参考

## 拍摄后立刻做

1. **当晚导电脑** → 按文件夹分类：`空间关系/` `校训石/` `图书馆/` `中流桥/` `天心湖/` `一鉴桥/` `瑞樟路/` `路灯氛围/`
2. **选3-5张最关键的** → 立刻导入即梦跑测试：校训石夜景、中流桥全景、图书馆门口夜景
3. **不删废片** → 糊的过曝的也留着，AI生图有些场景不需要高清
