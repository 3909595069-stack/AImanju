## 🕳️ 8步避坑清单

每条都是真实踩过的坑。每条都配有"坑前症状"——在掉进去之前识别它。

| 步骤 | 坑 | 坑前症状 | 避坑方案 |
|:--|:--|:--|:--|
| **01 剧本** | 写太长了——30分钟脚本幻想用 AI 做出来 | "这个剧情太好了，再加一段……" | 第一篇严格控制 1-3 分钟成片。写完后砍掉 50% 篇幅再开工 |
| **02 角色** | 不固定角色就开始生图 | "我先随便出几张图看看感觉" | 先完成角色定妆卡（正面+侧面+全身），锁定后再进入下一步 |
| **03 场景** | 每个镜头换一套场景描述 | "这个镜头换个风格应该也没事" | 为每个场景建母代码卡，所有该场景的提示词都用同一段开头 |
| **04 分镜** | 直接写提示词跳过文字分镜 | "我脑子里有画面，直接出图就行" | 先用文字写完所有分镜（8要素），审完分镜逻辑，再开始生图 |
| **05 生图** | 所有镜头用同一个 prompt 只改景别 | "反正都是这个场景，prompt 差不多" | 每个镜头的 prompt 都要反映该镜头的独特内容。景别、角度、动作、光影必须逐镜头调整 |
| **06 视频** | 一段视频塞太多动作 | "让角色一边走一边回头一边说话" | 一段视频 = 一个动作。多个动作 = 多个视频片段 |
| **07 配音** | 用 AI 默认语气直接导出 | "听起来能听懂就行了" | 标注情绪提示词 + 调整语速 + 对比真人参考音频 |
| **08 发布** | 随便写个标题就发布 | "内容好就行了，标题不重要" | 至少写 3 个候选标题，选最有好奇心缺口的那个。封面必须单独设计 |

## 📖 推荐学习资源

### 视频教程（B站）

| UP主/频道 | 内容方向 | 适合阶段 |
|:--|:--|:--|
| 搜索"即梦教程" | 即梦全流程实操——角色设计到图生视频 | 入门到进阶 |
| 搜索"AI 短片" | 完整 AI 短片制作过程——看别人从头到尾怎么做 | 入门 |
| 搜索"电影分镜拉片" | 经典电影分镜分析——培养导演思维 | 进阶 |
| 搜索"剪映进阶教程" | 关键帧动画、调色、转场、蒙版 | 剪辑阶段 |
| 搜索"电影调色分析" | 光影、色彩、情绪——视觉语言 | 进阶到高级 |

### 社区与平台

| 平台 | 内容 | 使用建议 |
|:--|:--|:--|
| **即梦社区** | 即梦用户分享的 prompt 和作品，直接可复用 | 每天刷 10 分钟——看到好的 prompt 复制下来，备注"哪种效果" |
| **小红书** | AI 短片教程、prompt 分享、即梦/可灵使用技巧 | 搜索"AI 动画""AI 短片"，大量创作者在分享实操 |
| **Runway 官方社区** | Runway Academy 教程 + 用户作品展示 | runwayml.com/academy——英文但内容质量极高 |
| **Civitai** | SD/ComfyUI 模型和 LoRA 下载 | 需要 ComfyUI 时来这找模型 |
| **Discord** | Midjourney / Runway 官方 Discord | 看别人的 prompt 和生成结果，学 prompt engineering 最快 |

### 参考网站

| 网站 | 用途 | 说明 |
|:--|:--|:--|
| **[[film-grab.com]]** | 电影逐帧截图库 | 按电影浏览每一帧，学习构图、光影、色调——最被低估的免费资源 |
| **ArtStation** | 概念设计作品 | 搜索 `environment concept art` 获取场景参考 |
| **Pinterest** | 氛围板搭建 | 建自己的 Board，按场景类型收集参考图 |
| **Shotdeck** | 电影画面数据库 | 专业级，需付费。搜索特定情绪/色调/构图的电影画面 |

### 书籍推荐

| 书名 | 作者 | 读什么 |
|:--|:--|:--|
| **《电影语言的语法》** | 丹尼艾尔·阿里洪 | 分镜圣经——景别、角度、运镜、场面调度的底层逻辑。AI 短片也需要这些 |
| **《故事》** | 罗伯特·麦基 | 剧本创作的终极教科书——不是教你写"格式"，是教你写"好故事" |

> **学习路线建议**：先 B站看 3-5 个完整的 AI 短片制作视频 → 动手做第一部（哪怕只有 30 秒） → 遇到问题回来查本知识库 → 精读《电影语言的语法》→ 做第 2-5 部 → 看《故事》→ 做到第 10 部。
